🇯🇵 名古屋

Time：2025/02 ✈️ 航班與入境 去程：TPE → NGO 回程：NGO → TPE 此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code 🗓️ 行程 Day1｜桃園機場 - 名古屋中部機場 - 名古屋車站逛街 桃園機場 → 中部機場 → Check in → 逛街 → 🍽️ 名尺燒肉伏見屋飛驒牛別邸 → 唐吉軻德 → 🏠 HOTEL RESOL NAGOYA Note 下飛機先把入境QR碼準備好，大概20分鐘出關，10分鐘等行李加安檢，整體大概半小時。然後直走大約10分鐘就會到名鐵名古屋機場站。接著搭乘大約37分鐘後抵達名古屋車站，Check in之後就到名古屋車站的百貨購物UQ跟GU拉!!! 然後晚餐去吃名尺燒肉伏見屋飛驒牛別邸，這間得肉質真的好讚，吃完都有點意猶未盡的感覺。 接著大約走路15分鐘到唐吉軻德買草莓、綠葡萄跟零食餅，車站附近這間比較小型，僅有生鮮雜貨。 Day2｜合掌村一日遊 名古屋車站集合 → 合掌村 → 飛驒高山老街 → 🍽️ 小雞蛋糕 → 🍽️ 世界的山醬 → 🏠 HOTEL RESOL NAGOYA Note 第二天是參加Klook一日團，配合得當地旅行社是GOGODAY，商家前一晚會將導遊聯絡資訊、集合相關資訊等再寄發Email給旅客，一早導遊會在集合地點舉旗集合大家，因為當日前往人數較多，導遊有微調行程順序，但是每個景點還是都有如期去到。 第一站是先到合掌村展望台眺望合掌村及拍照留念，接著是到合掌村村落去逛街，可以近距離觀看合掌村的房子結構，這邊也有販賣明信片跟郵筒，可以在這邊寄出明信片！當天因為天氣很好，有些屋頂開始在融雪，但整體來說逛逛還蠻舒服的！ 接著是到飛驒高山老街，我們先去買了飛驒牛壽司吃，這間會有仙貝當作底盤裝壽司給客人，也有室內用餐區可以坐著品嚐壽司。接著去高山布丁亭，有蘇打口味布丁蠻酷的，喜歡品酒的話，這間店：山車，可以購買小酒杯，並且試飲整櫃的清酒。 接著回程到名古屋車站後，剛好經過小雞蛋糕，只有白天時段可以外帶，晚上只有內用，低消是一杯飲料，小雞蛋糕外型可愛，吃起來整體也很搭。然後我們去吃有名的炸雞：世界的山醬，一般基本款是沒加辣但是有胡椒，若是選減辣則會變成刷醬口味。 Day3｜名古屋城 - 大須觀音寺/商店街 - Urban Quar 名古屋城 → 🍽️ 山本屋總本家 → 大須觀音寺/商店街 → 榮商圈逛街 → 綠洲21 → Urban Quar泡湯 → 唐吉軻德 → 🍽️ 一蘭 → 🏠 HOTEL RESOL NAGOYA ...
