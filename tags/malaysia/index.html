🇲🇾 蘭卡威

Time：2024/06 ✈️ 航班與入境 去程：TPE → SIN → LGK 回程：LGK → KUL → TPE 此次旅程時新加坡與馬來西亞皆免簽 72小時內上網填寫【新加坡電子入境卡SG Arrival Card】 🗓️ 行程 Day1｜桃園機場 - 樟宜機場 - 蘭卡威機場 - 訂跳島行程 - Hidden lankawi - 按摩 - 夜市 桃園機場 → 樟宜機場 → 蘭卡威機場 → 飯店 → 訂行程 → 買伴手禮 → 🍽️ 晚餐Hidden lankawi → 按摩 → 🍽️ 夜市 → 🏠 CENANG PLAZA BEACH HOTEL Note 搭紅眼班機抵達新加坡後有大約5個半小時等待轉機，我們決定入境新加坡去逛星耀樟宜。因為T1轉機區的亞坤在整修，所以我們是入境後先到T3吃亞坤，接著十點半左右去看瀑布&逛逛百貨公司，最後去B2吃松發肉骨茶，就出境回到候機室了(出境時只有快速通關刷護照、檢查行李跟身份是在登機口)。 蘭卡威機場蠻小的，抵達後排隊查驗護照大約20分鐘左右，接著我們在機場門口與提早於易遊網預定的機車行碰面去牽機車(易遊網只告訴我們會有人舉牌接我們，我們到門口後完全無法與對方聯繫上，後來是其他商家協助聯繫的)。機場大廳有很多商家會問你要不要租車，價格也比較便宜，大家也可以考慮到當地後直接租車。 放完行李後走到附近的AB Tour在AB Motel門口，我們預定了跳島行程(孕婦島、巨獅島、溼米島)。因為距離日落還有一點時間，所以我們先到飯店附近的伴手禮店買了一些歐米呀給，蘭卡威整座島都是免稅的，所以買伴手禮蠻便宜的。 騎去Hidden lankawi吃晚餐看日落，這家餐廳在沙灘旁邊，可以直接走下去沙灘、也有駐唱歌手、座位有戶外跟室內，雖然都沒有冷氣，但是傍晚去的時候不會很熱～～ 吃飽後到ALUN-ALUN spa，這是我們先在KLOOK上預定的，不過預定時沒讓選店家，但蘭卡威有四五間分店，所以我們還是到現場詢問是否有時段。選擇1小時的精油按摩，我跟璇按起來的感覺是按摩師只是順著肌肉推油，沒有特別有感，而且按摩師還一直講悄悄話嬉鬧。 ...
