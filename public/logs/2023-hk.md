# 🇭🇰 香港

#### Time：2023/06

## ✈️ 航班與入境
> 去程：TPE → HKG </br>
回程：HKG → TPE </br>
此次旅程時使用台胞證入境


## 🗓️ 行程
###  Day0｜桃園機場 - 香港機場
香港機場 → 🏠佐敦airbnb

<details>
<summary>Note</summary>

因為是搭晚班機，所以抵達香港時已經晚上11點多，先購買八達通及儲值後，就跟著巴士的指示牌前進，雖然時間已經接近半夜，但是巴士站還是人超多～大概搭乘50分鐘後抵達住宿附近。
</details> 

###  Day1｜迪士尼
🍽️ 澳洲牛奶公司 → 迪士尼 →🍽️ 興記煲仔飯 → 🏠佐敦airbnb

<details>
<summary>Note</summary>
  
早上先去吃澳洲牛奶公司，雖然隊伍很長，但是大概排隊15分鐘左右就入座，啊來香港是肯定要併桌吃飯的～～點了有名氣的通心粉、炒蛋、燉奶，大概30分鐘內就可以用餐完畢。
	
接著搭地鐵去迪士尼站，10:00開園(很貼體讓大家睡飽的迪士尼哈哈哈哈)，第一站當然是先前往禮品店購物喜歡的髪箍～～接著遊玩順序是：明日世界/蟻俠與黃蜂女擊戰特攻 → 12:30-13:00探險世界/獅子王慶典 → 灰熊山谷/灰熊山極速礦車 → 迷離莊園/迷離大宅 → 反斗奇兵/玩具兵團跳降傘 → 幻想世界/米奇幻想曲(10m) → 小小世界 → 小熊維尼歷險之旅 → 20:30-20:50煙火。
	
結束後搭地鐵回市區吃興記煲仔飯當晚餐&宵夜。
</details> 

###  Day2｜旺角 - 中環 - 九龍
🍽️ 一點心 →🍽️ 金華冰廳 → 金魚街 →🍽️ 妹記生滾粥 → 波鞋街 → 旺角 → 女人街 →🍽️ 富豪雪糕車 → 天星碼頭 → 搭船 → 中環 → 太平山纜車 → 山頂凌霄閣 → 蘭桂坊 →🍽️ 十八座狗仔粉 →🍽️ insomnia → 尖東海傍散步 →🍽️ 香港仔一品魚蛋王 → 🏠佐敦airbnb

<details>
<summary>Note</summary>
    
早午餐前往一點心用餐，抵達時外面已經有些人潮排隊，大約排了半小時後入座，就是間經典的港式餐廳，是點餐送過來的(沒有推車自取蒸籠)。接著到金華冰廳外帶菠蘿油跟港奶，個人覺得應該內用會更美味！然後逛一下金魚街，就是很多店家外面都會掛一袋一袋的金魚。雖然很熱，但我們還是去吃妹記生滾粥，粥很好吃，但夏天來吃真的很熱，會邊吃邊狂流汗哈哈哈哈。接著是到波鞋街，這邊真的有很多家運動用品店，但好像沒有特別便宜。女人街也在附近可以一起走走看看，但沒有特別好買就是了。不過沿路遇到一家魚蛋威威小食還蠻好吃的。
	
接著準備到天星碼頭搭船過去中環，剛好遇到富豪雪糕，就排隊一下，拿到要趕快吃，不能拍照太久，因為很快就會融化，吃起來大概就像蛋捲冰淇淋的味道～
	
到中環碼頭後，可以搭叮叮車到太平山纜車站山腳下，出示線上購買的QR掃描後，就可以排隊等搭纜車，建議是坐車廂的右邊，因為右邊面對風景，左邊面對山壁。在山頂待到天黑後，就準備下山，但這時候下山的纜車已經大排長龍，所以我們到巴士站去排隊，但也排了40分鐘才上公車前往蘭桂坊。
	
蘭桂坊這時已經開始熱絡起來，我們先隨意進入餐廳十八座狗仔粉填飽肚子後，前往insomnia九巴喝酒，這家酒吧有現場live，也有人在中間隨歌起舞，整體氣氛還不錯！
	
大約半夜12點左右我們就到搭地鐵回到尖沙咀附近的維多利雅港邊散步～然後再散步回飯店，路上又經過一家香港仔一品魚蛋王就進去吃了一頓宵夜～

</details> 

###  Day3｜尖沙咀 - 香港島 - 香港機場 - 桃園機場
🍽️ 尖沙嘴蘭芳園 → 買珍妮曲奇 → k11 → 🍽️ five guys → 維多利亞港 → 搭船 → 香港島 → 🍽️ 甘牌燒鵝 → 香港機場 → 桃園機場

<details>
<summary>Note</summary>
    
早上到蘭芳園尖沙咀店用餐，大概排隊15分鐘左右就入座併桌用餐，很好吃，請一定要來吃！接著到隔壁大樓買珍妮曲奇當伴手禮，然後到k11裡逛逛吃five guys漢堡後就搭船回到香港島外帶甘牌燒鵝(感覺內用會更好吃，希望下次有機會！)，接著就搭機場快綫到機場啦～
</details> 




###  總結
> 六月去香港真的太熱了，下次應該選涼一點的月份。香港的東西都好好吃，但是物價不便宜！地鐵、公車都很方便。香港迪士尼偏小，如果是平日去的話，應該不用買快速通關，城堡煙火秀很漂亮，請一定要留下來看！
> 
> 香港的餐廳都沒有提供免費衛生紙，可以自己準備或是跟餐廳購買，然後因為很多名店都會人潮擁擠，所以大致上都是會跟陌生人拼桌的。
