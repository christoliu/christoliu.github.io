🇯🇵 北海道

Time：2022/12 ✈️ 航班與入境 去程：TPE → CTS 回程：CTS → TPE 此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code 🗓️ 行程 Day1｜桃園機場 - 新千歲機場 - TOMAMU 桃園機場 → 新千歲機場 → 南千歲JR站 → TOMAMU車站 → 星野接駁車接送至飯店→ 🍽️ Soup King Curry → 微笑海灘 → 🍽️ 森之餐廳 → 愛絲冰城 → 水之教堂 → 林木之湯 → 🏠 TOMAMU Note 抵達新千歲機場後，就直接前往巴士搭乘處搭巴士到南千歲等搭JR，巴士下車處要走一個空橋到南千歲JR站，大約搭乘1.5h後抵達TOMAMU站，會有星野的接駁巴士在這邊等大家。 飯店check-in後，就可以搭度假村裡面的接駁巴士前往各景點拉！我們先晃到美食區去吃Soup King Curry，蠻好吃的，價格也不算太貴！吃飽後去微笑海灘玩水，裡面很溫暖有人造海浪跟spa池，跟外面的冰天雪地很反差～ 晚餐到森林餐廳去用餐，餐點我們個人覺得普通，但是餐廳的氛圍感很好，所以還是推薦去吃！愛絲冰城有很多冰塊做的房子跟溜冰場，但我們來的時候溜冰場還沒弄好，所以就先在冰塊屋吃東西烤棉花糖，然後晚上這邊也會放音樂跟煙火(要先查一下時刻表)。接著走到水之教堂去參觀。 林木之湯很推薦大家去泡，是男女分開的露天風呂，身體泡在45℃熱水，頭在-15℃欣賞森林雪景 Day2｜TOMAMU 霧冰平台 → 滑雪 → 🍽️ Jingisukan KING → 愛絲冰城 → 林木之湯 → 🏠 TOMAMU ...
