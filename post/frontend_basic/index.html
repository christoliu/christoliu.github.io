HTML

包含了一系列的元素(Elements)，而元素包含了標籤(Tags)與內容(Content)、屬性(Attribute)。

<p class="note">My cat is very grumpy.</p>

Element：<p class=“note”>My cat is very grumpy</p> Tags：<p></p> (Opening tag / Closing tag) Contant：My cat is very grumpy. Attribute：class

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
    </head>
    <body>
        <div>
            <h1><span></span></h1>
            <p><a></a></p>
        </div>
    </body>
</html>

:::info <!DOCTYPE html>：文件類型（doctype）HTML5 <html></html>：<html> 元素，又被視為根元素（root element） <head></head>：<head> 元素，裡面放的是你想涵括的重要資訊，但不會顯示於網頁瀏覽者眼前的 <body></body>：<body> 元素，包含了所有會顯示於網頁瀏覽者眼前的內容 <title></title> — 呈現於網頁瀏覽者眼前的網頁標題。 <meta charset=“utf-8”>：指定文件使用utf-8這種字元編碼 :::

一種風格頁面語言(style sheet language)，讓你在 HTML 文件中的元素(element)上套用不同的頁面樣式(style)。

//HTML
<link href="style.css" rel="stylesheet" type="text/css">

p .class_name {
    color: red;
    /*font-size: 100px*/

:::info Selector：p、class_name Declaration：color: red; Property(屬性)：color Property value：red 註解：/* */ 內的文字 :::

  • Selector可選擇：

    • Element
    • Class(.class_name)
    • Attribute(img)
    • Pseudo-class(a:hover)
    • ID(#my-id)

  • 排版

    • margin: 元素==外部==的空間
      • 向外推
    • border: 位於矩形==內容==外部的實線
      • 向內推
    • padding: 位於==內容==周圍的空格
      • 向內推
      • ex:段落文字周圍

  • css reset

    • Normalize：保留一些預設樣式
    • Reset：完全無保留預設樣式
      • img reset(見RWD)
      • block reset

        • 簡化排版算數

    • id主要作為錨點、後端js
    • style：動態樣式時會使用
    • .html讀入css的順序也可以覆蓋

RWD

  • CSS media
    • 於偵測到設備寬度低於設定px以下會被開啟
    • 會覆寫前面的相同設定
    • 永遠不要寫死X軸設定 (width: px;)
      • 造成網頁需左右滑
      • 解決方法：
        • max-width
        • %
        • 設定最大值，可以依照父層大小自適應
    • img 自適應==reset==
      • 防止載入圖片像素過大
      img{
    max-width: 100%;
    height:auto;
}
    • 記住客戶的載具、瀏覽器、設計稿使其剛好符合
    • 斷點：
      • media的個數
        • 含最原始
        • 越少越好
        • 通常為3(iphone、ipad、macbook)
@media(max-width: 1140px){/*for PC*/
    .header{
        max-width: 100%;    
    }
}
@media(max-width: 768px){/*for ipad*/}
@media(max-width: 375px){/*for iphone*/}

    • TCP/IP：被視為 OSI 七層協定的簡化版，1990 年開始蓬勃發展(www/mail)。

    • 或 ( or )：ip.src== xx.xx.xx.xx or ip.dst == xx.xx.xx.xx
    • port： tcp.port == 80
    • 網路請求：http.request.method==“POST”
