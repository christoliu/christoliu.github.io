🇯🇵 福岡

2 min

Time：2025/05#

✈️ 航班與入境#

去程：TPE → FUK
回程：FUK → TPE
此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code

🗓️ 行程#

Day1｜桃園機場 - 福岡機場 - 天神地下街 - Mina百貨 - 福岡塔#

桃園機場 → 福岡機場 → 🍽️ MIGNON可頌 → Check in → 🍽️ 元祖博多明太子飯(天神南站) → 🍽️ I’m donut?(天神南站) → 天神地下街 → Mina百貨(天神站) → 福岡塔(西新站) → 🍽️ 博多牛腸鍋 山中(赤坂站) → 🏠 博多WBF格蘭德飯店

Note

機場國際線出關很快，大約20分鐘就過完海關+檢疫刷護照，會先搭接駁車前往國內線，也就是地下鐵入口。一天會搭超過三趟，建議從機場出發就可以買一日票。

抵達博多站因為很餓，所以先買了MIGNON可頌墊墊肚子，接著到飯店去寄放行李，然後展開福岡之旅。第一站先去吃明太子飯，上餐速度很快，建議可以點明太子飯跟沾麵套餐，兩種口味交換吃比較不會太膩。

接著去排I’m donut?，大概排15分鐘就入店，排隊動線是不能走回頭路的，所以建議可以先上網看好自己要購買的口味，雖然店員說要當日食用完畢，但我放到隔天早上吃一樣好吃！

天神地下街有很多小店，可以從天神南連通到Mina百貨，因為這裡有UQ、GU、Loft，所以可以消磨不少時間XD

接近傍晚時我們在福岡塔一樓現場購票後，直接搭電梯上塔，建議是黃昏附近來，可以看到日景跟夜景！

牛腸鍋出乎意料好吃，牛腸很Q又很香，完全沒臭味，整鍋裡面大概會有蔬菜、牛腸、豆腐，可以再加點食材，不過觀察了一下隔壁桌，似乎都是當下酒菜，邊喝酒邊吃，不是當正餐吃。

回飯店後，我有到浴場看了一下，小小一間大概容納10位左右就滿了，所以就沒去泡湯。

Day2｜櫛田神社 - 中洲川端通商店街 - B･B･B POTTERS - 博多運河城#

🍽️ Dacomecca(博多站) → 🍽️ The Full Full Hakata(櫛田神社站) → 櫛田神社 → 中洲川端通商店街 → 🍽️ Fuk Coffee(櫛田神社站) → B･B･B POTTERS(藥院大通站) → 博多運河城(櫛田神社站) → 🍽️ かわ屋 → 🍽️ 中洲關東煮 → 🏠 博多WBF格蘭德飯店

Note

Dacomecca入內後，先挑選喜歡的麵包，一樣是不能走回頭路的動線，所以看到喜歡的就要先拿，內用的話低消是一杯飲料，然後可以到座位區找位置享用，我看大家都一次買5、6個麵包吃XDD

接著到The Full Full Hakata買明太子麵包，因為平日加上下大雨，所以完全沒有排隊，進店後直接到櫃台點餐結帳！明太子醬很香，麵包也很好吃！

櫛田神社小小的，大概15分鐘可以繞完一圈，中洲川端通商店街是有屋頂的商店街，所以下雨的話很適合來這邊逛街躲雨！我們是從櫛田神社這邊逛到中洲川端，出去走5分鐘就有很大間的唐吉軻德可以逛！

接著到Fuk Coffee喝下午茶，這家是要點餐後，才找位置入座，店內空間不大約10桌左右。然後等雨小一點，我們去逛B･B･B POTTERS這間大部分是廚具相關，但也有可愛小物可以逛！

博多運河城沒有我常逛的品牌，但是室內逛街加上水舞秀，還是可以排入行程去走走看看。かわ屋感覺是當地人吃的居酒屋，裡面幾乎都是日文聊天的客人，我們是坐在吧台的位置，點的每串串燒都沒有踩雷，蠻推薦的！

因為下整天雨，所以沒有屋台料理出來擺攤，查到這家店面的中洲關東煮，位置大概10個，所以我們大約等了半小時才進店，菜單只有日文跟韓文，只收現金，當天遇到蠻多韓國人來吃的。湯頭很讚，關東煮也都很入味，蠻推這間店的！

Day3｜柳川遊船 - 太宰府天滿宮#

西鐵福岡駅 → 🚈 天神-柳川 → 柳川遊船 → 🚈 柳川-二日市-太宰府 → 太宰府天滿宮 → 🍽️ 一蘭 → 🍽️ 草莓最中大福 → 🍽️ 梅枝餅かさの家 → 🚈 太宰府-二日市-福岡天神 → 大創(博多站) → 🍽️ Hakata Yakiniku Yasohachi Chikushiguchi → 🏠 博多WBF格蘭德飯店

Note

西鐵福岡駅2樓的遊客中心可以買柳川/太宰府套票，直接走進遊客中心比選單上的數字就可以了，工作人員會跟你一大張車票跟折價券，領取後，出事票券走西鐵福岡駅人工通道，就可以到3號月台等車。這個套票只能買當天的，可以刷卡，福岡往柳川的特急車是00跟30分有車！

從福岡直達柳川大約搭50分鐘，下車後才收票，出口會有工作人員發地圖，也會指引你到接駁車搭車處，接駁車會直接載你到柳川搭船處，出示票券後，就可以排隊上船，如果本身沒帶帽子，建議拿斗笠，因為整趟船程大約1小時，沿路沒有什麼遮蔭處，會蠻曬的。

搭完船後，也有接駁車將你載回搭船處或柳川站，從柳川前往太宰府要在二日市轉乘，指標蠻清楚的！抵達太宰府站後，出口就是一蘭，建議可以先去抽號碼牌，我們當天大約3點抽，然後就先去逛商店街，等到約4點才入座，吃完出來4:30就已經不給抽號碼牌了！

草莓最中大福這家蠻夯的，很快就會售完，如果經過有想吃就趕快先買！梅枝餅 かさの家是現場邊製作，所以應該不會售完，可以最後買了帶走也可以。

太宰府其實沒有地圖上看起來那麼大，逛一圈拍照大概40分鐘可以結束，然後這邊的商店街都差不多5點就開始收攤了，所以有喜歡的東西建議要注意時間購買。

回市區後先到博多站的大創採購一番後，本來要去吃Nikuichi燒肉，但現場要等3小時，只好臨時換店去Hakata Yakiniku Yasohachi Chikushiguchi，座位是要脫鞋類包廂的感覺，如果懶得單點也可以直接點套餐！

Day4｜福岡機場 - 桃園機場#

福岡機場 → 桃園機場

Note

大約是起飛前2小時到國內線等接駁車，接駁車大約5～10分鐘一般，開過去國際線大概8分鐘車程。可能因為一大早飛機班次不多，所以拿到票、安檢、過海關大概只花了20分鐘，進到管制區大概是9點，免稅店就都已經開了！重點是有Onitsuka Tiger鬼塚虎，但款式跟福岡市區的比較不同。

總結#

四天其實蠻剛好作為一個小旅行，五月去天氣很合適，涼涼的加上太陽就是不會流汗舒適的天氣。建議可以住離博多站越近越好，因為購物後不用走太遠回飯店，畢竟日本都是每天2萬步起跳XD

博多的阪急站B1可以買伴手禮！AMANBERRY、Sugar Butter Tree兩櫃就在旁邊，建議早點去AMANBERRY蠻夯的，那天大約10點左右，有幾種尺寸就已經完售了，然後這裡是合併計算發票有¥5500就可以拿到樓上退稅櫃台直接退稅！
