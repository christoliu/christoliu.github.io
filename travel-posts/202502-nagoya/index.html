🇯🇵 名古屋

1 min

Time：2025/02#

✈️ 航班與入境#

去程：TPE → NGO
回程：NGO → TPE
此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code

🗓️ 行程#

Day1｜桃園機場 - 名古屋中部機場 - 名古屋車站逛街#

桃園機場 → 中部機場 → Check in → 逛街 → 🍽️ 名尺燒肉伏見屋飛驒牛別邸 → 唐吉軻德 → 🏠 HOTEL RESOL NAGOYA

Note

下飛機先把入境QR碼準備好，大概20分鐘出關，10分鐘等行李加安檢，整體大概半小時。然後直走大約10分鐘就會到名鐵名古屋機場站。接著搭乘大約37分鐘後抵達名古屋車站，Check in之後就到名古屋車站的百貨購物UQ跟GU拉!!!

然後晚餐去吃名尺燒肉伏見屋飛驒牛別邸，這間得肉質真的好讚，吃完都有點意猶未盡的感覺。

接著大約走路15分鐘到唐吉軻德買草莓、綠葡萄跟零食餅，車站附近這間比較小型，僅有生鮮雜貨。

Day2｜合掌村一日遊#

名古屋車站集合 → 合掌村 → 飛驒高山老街 → 🍽️ 小雞蛋糕 → 🍽️ 世界的山醬 → 🏠 HOTEL RESOL NAGOYA

Note

第二天是參加Klook一日團，配合得當地旅行社是GOGODAY，商家前一晚會將導遊聯絡資訊、集合相關資訊等再寄發Email給旅客，一早導遊會在集合地點舉旗集合大家，因為當日前往人數較多，導遊有微調行程順序，但是每個景點還是都有如期去到。

第一站是先到合掌村展望台眺望合掌村及拍照留念，接著是到合掌村村落去逛街，可以近距離觀看合掌村的房子結構，這邊也有販賣明信片跟郵筒，可以在這邊寄出明信片！當天因為天氣很好，有些屋頂開始在融雪，但整體來說逛逛還蠻舒服的！

接著是到飛驒高山老街，我們先去買了飛驒牛壽司吃，這間會有仙貝當作底盤裝壽司給客人，也有室內用餐區可以坐著品嚐壽司。接著去高山布丁亭，有蘇打口味布丁蠻酷的，喜歡品酒的話，這間店：山車，可以購買小酒杯，並且試飲整櫃的清酒。

接著回程到名古屋車站後，剛好經過小雞蛋糕，只有白天時段可以外帶，晚上只有內用，低消是一杯飲料，小雞蛋糕外型可愛，吃起來整體也很搭。然後我們去吃有名的炸雞：世界的山醬，一般基本款是沒加辣但是有胡椒，若是選減辣則會變成刷醬口味。

Day3｜名古屋城 - 大須觀音寺/商店街 - Urban Quar#

名古屋城 → 🍽️ 山本屋總本家 → 大須觀音寺/商店街 → 榮商圈逛街 → 綠洲21 → Urban Quar泡湯 → 唐吉軻德 → 🍽️ 一蘭 → 🏠 HOTEL RESOL NAGOYA

Note

早上出發到名鐵站購買一日車票，車站的自動販賣機就能購買，接著搭車前往名古屋城，名古屋城外面有一排商店街，可以到這邊用餐。我們抵達後先到山本屋總本家(味噌烏龍麵名店)吃午餐，接著才前往名古屋城，可以出示名鐵一日票，門票¥500可以折扣¥100。

在城內逛逛後，前往大須觀音寺參觀與大須商店街逛街吃吃喝喝！大須商店街推薦小食店：Fruits candy ar 吃已切塊方便食用的糖蘋果、甘香坊喝純濃的抹茶/福吉茶那提、Nigiritate Osu Banshojidori 吃各種口味的三角飯糰～

接著搭車前往名鐵榮站，出站後附近有鬼塚虎、大創、NB、唐吉軻德等等各種商店，這站非常適合逛街購物～～接著傍晚前往綠洲21觀景台，免門票，簡單散步看一下風景大約安排20分鐘左右即可。接著在名鐵榮站可以搭乘每小時一班的接駁車前往Urban Quar泡湯(可以到官網看班次表)！

Urban Quar有岩盤浴、室內外温泉、休憩區、餐廳，購買3小時卷，大約台幣260。進去後會先放鞋子然後領到一個磁扣，接著買入場票、裡面用餐也都是使用該磁扣紀錄費用，接著到櫃檯領取岩盤浴及泡湯的服裝、毛巾，就可以上樓換衣服。岩盤浴區是需要著店家給的簡易服裝且沒有分性別的，泡湯區就有區分男湯、女湯的入口，進入後會先到更衣區，這邊有可以上鎖的衣物櫃，可以將身上所有物品都鎖在此處，接著入場是先到室內區，有各式不同的湯種，也有烤箱、蒸氣室，然後樓梯是通往頂樓得露天溫泉區！泡湯後，盥洗區也有附盥洗用品。最後離開前要到自助結帳機器掃手上的磁扣就可以離開拉～

離開後我們末班接駁車回到名鐵榮站，前往唐吉軻德購物，這間雖然商品更多，但是沒有生鮮區，所以這家是買不到水果的，然後因為稍微大間一點，所以退稅櫃台通常都大排長龍，大概要排20分鐘排隊免稅結帳。

結束一天行程後，我們回飯店放東西後去吃一蘭，完全不用排隊的感覺真好～～

Day4｜JR Gate Tower逛街 - 名古屋中部機場 -Flight of dreams - 桃園機場#

Coffee House Kako → JR Gate Tower逛街 → 中部機場 → Flight of dreams → 桃園機場

Note

大約08:20抵達店家Coffee House Kako，前面排隊四組人，大概等候半小時才入內用餐，是一家很可愛的小店，早餐的配置就是選一杯飲品加上一份店家自製果醬吐司的組合，土司烤得很香很好吃，坐在吧台還能看到店員製作餐點的過程，店內散發咖啡香與吐司的香氣～很推薦大家來這間店！不過這家店只收現金喔。

接著到JR Gate Tower的郵局買特殊款明信片後，就離開名古屋前往機場拉！機場二航廈有Flight of dreams展區，展示波音787飛機，以及兒童玩耍區，等飛機的時間可以到這邊逛逛喔！

總結#

四天其實蠻剛好作為一個小旅行，雪季去合掌村的話還是建議跟團比較安全。
Urban Quar的泡湯環境很不錯，下次會回訪。
