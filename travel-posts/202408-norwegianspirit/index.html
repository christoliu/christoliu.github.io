🚢 挪威郵輪奮進號

1 min

Time：2024/08#

行前#

  1. 雄獅官網報名
  2. 宮古島選擇報名雄獅的一日團或是自行上網預定包車自由行
  3. 收到說資包(船票、出團手冊、行李條、船卡吊牌、預辦登船說明、優惠券)
  4. 72小時前線上預辦登船
  5. 出發當天記得攜帶！護照、說資包！

航程#

基隆港 → 沖繩-宮古島 → 基隆港

🗓️ 行程#

Day1｜基隆港務大樓報到#

基隆港務大樓 → 托運行李 → 11:00櫃檯報到 → 12:00開始出境檢查 → 人員收護照 → 登船 → 🍽️ 午餐Window → 🍽️ 下午茶The Local → 🍽️ 晚餐Silk → 🍽️ 宵夜The Local

Note

登船後我們直接到Window吃午餐，接著到隨意晃晃等開船，三點的時候會廣播通知請大家至7F甲板聆聽救生衣使用方式，接著三點半在12樓三角帆酒吧有一個迎賓Party，船員會與賓客一起跳舞～～然後就開始各種參觀船上設施！晚上會有人員將護照影本及海關資料表到各艙房，以便大家明早下船。

Day2｜宮古島自由行#

🍽️ 早餐Garden → 宮古島自由行 → 船上泳池玩水 → 🍽️ 下午茶Garden Cafe → 休息 → 🍽️ 晚餐Taste

Note

宮古島可以選擇參加旅行社提供的付費一日團(優先下船，有遊覽車接駁)，我們是決定下船後自行晃晃

一大早船就會靠岸宮古島，並且開放賓客攜帶護照影本、黃色的海關申報單、房卡，依序下船，下船後會先搭接駁車至宮古島海關邊境檢查處，查驗護照影本與背包物品後，就入境沖繩-宮古島拉！外面就是公車、計程車及雄獅團上車處。因為公車班次時間要等比較久，所以我們選擇搭計程車。

第一站我們先到唐吉軻德和AEON大採買一波，接著搭公車路線C回到海關邊境檢查處(公車一人500日幣)，然後大約走5鐘去吃沖繩有名的蕎麥麵Minato Shokudou，這家好推!!!吃完就回到海關邊境檢查處，檢查隨身物品後，出境搭接駁車返回船上

Day3｜回基隆港#

🍽️ 早餐Garden → 🍽️ 早餐Window → 下船 → 海關查驗入境 → X光檢查物品

Note

看日報有寫說5:30-6:00Garden有提供歐式早餐，所以特別早起，結果差別只是把一般時段裡的麵包先擺出來給大家吃，沒有什麼特別的，所以等到7:00我們又到Window去吃早餐，發現大部分也都跟Garden有的菜色差不多。接著8點多就排隊下船了，整體流程蠻順暢的。

總結#

建議必備暈船藥，如果是平時容易暈車的人，可以開船前先服用。第一天可以準時11點就報到，這樣可以拿到順序比較前面的登船號碼、登船時可先到7樓服務台預約Silk亞洲餐廳。第二天宮古島因為島上計程車數量較少，所以我們9點下船，10點開始排大概等到10:40，大家可以考慮9:30再下船，這樣計程車比較不會排那麼久。第三天建議7點多離開房間，攜帶行李到喜歡的餐廳吃早餐等下船。

💰 費用#

  • 團費：TWD23880 / 雙人 / 三天兩夜宮古島 /內艙房(無窗)，含設施使用跟免費餐廳
  • 郵輪小費：USD20 人 / 晚
