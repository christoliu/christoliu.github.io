🇯🇵 東京-春

2 min

Time：2024/03#

✈️ 航班與入境#

去程：TPE → HND
回程：NRT → TPE
此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code

🗓️ 行程#

Day0｜桃園機場 - 羽田機場#

桃園機場 → 羽田機場 → 🍽️ 宵夜全家 → 🏠 plat hostel keikyu haneda home

Note

去程搭的時候紅眼的樂桃，所以選在羽田機場附近住宿。

Day1｜TOMAMU - 六本木新城 - 新宿御苑 - 銀座#

穴守稻荷站 → 大門站 → Check-in放行李 → 銀座站買一日車票 → 🍽️ 午餐肉源燒肉 → 六本木新城 → 🍽️ 點心matsunari和菓子 → 新宿御苑 → 新宿 → 🍽️ 咖啡blue bottle → 銀座 → 🍽️ 晚餐butaball居酒屋 → 🏠 Super Hotel 浜松町

Note

肉源燒肉我們大概11:20左右抵達，沒有排隊就直接入座，價格不貴，CP值蠻高!!吃完後先到六本木新城閒逛看東京鐵塔，接著去新宿御苑看櫻花，本來跟著櫻前線預測買的機票，結果前一週突然來了韓流，所以櫻花沒有看得很滿QQ，不過晴天到這邊看看花、散步、躺在草皮上曬太陽蠻舒服的。

Day2｜富士山一日遊#

🍽️ 當勞鬆餅堡 → 東京車站集合(kkday富士山一日遊) → 日川時計店 → 新倉山淺間神社 → 忍野八海 → 河口湖大石公園 → 🍽️ 晚餐東京車站電光石火廣島燒 → bic camera、唐吉軻德 → 🏠 Super Hotel 浜松町

Note

今天是買kkday的一日遊，一早先在東京車站集合上車，推薦可以去東京車站裡面的麥當勞外帶鬆餅堡當早餐到遊覽車上吃。我們買的一日遊都是以看富士山為主，所以每個景點都是去看富士山，如果遇到天氣不好，富士山不賞臉，就有點可惜。三月底剛好還有草莓，所以吃完晚餐後就到超商去買又便宜又大顆又好吃的草莓。

Day3｜迪士尼#

迪士尼陸地 → 🍽️ 宵夜鳥貴族 → 🏠 Super Hotel 浜松町

Note

因為東京迪士尼陸地吸引我們的設施比較少，所以在這次我們大約9點才入園，一入原先去商店索取生日貼紙(推薦生日附近又來迪士尼的人可以找商家索取，基本上問了就有，不會查驗日期)，我們的遊玩順序是：白雪公主冒險旅程Snow White’s Adventures → 星際旅行：冒險續航Star Tours: The Adventures Continue → 美女與野獸 魔法物語 Enchanted Tale of Beauty and the Beast → 14:00-14:45迪士尼眾彩交融遊行 → 西部沿河鐵路 → 午餐火雞腿&米奇米漢堡&米奇手套刈包 → 幽靈公館 → 18:45-19:30夜幕彩輝(雨天的遊行) → 怪獸電力公司「迷藏巡遊車Monsters, Inc. Ride & Go Seek → 太空山Space Mountain → 20:30-20:35眾彩滿夜空。

Day4｜皇居 - 涉谷明治通 - 中目川 - 上野公園 - 成田機場#

皇居 → 🍽️ 早午餐bills ginza → 涉谷明治通賞櫻花 → 🍽️ 午餐afuri → 中目川 → 買福砂屋 → 上野公園 → 🍽️ 全家豬排三明治 → skyliner → 成田機場 → 飛機Delay → 🏠 成田東武飯店

Note

bills ginza的鬆餅很好吃，座位也很適合聊天，接下來就開始櫻花踩點的一天。

晚上已經到機場時才收到捷星航空的延誤通知Email，櫃檯只有發班機延誤證明，以及用一塊白板寫飛機改到隔天早上才飛，所以我們就趕快找機場附近的飯店入住，還好成田東武飯店還有房間，而且一樓也有Lawson可以簡單買食物吃！

Day5｜成田機場 - 桃園機場#

總結#

若是要追櫻花可以參考櫻前線的預測，我們大概是12月底才買的機票，機票跟住宿價格都還行，不過就是有可能會有小插曲，像是這次就突然遇到寒流來，櫻花比預測的晚一週開。

klook的富士山一日遊也蠻推薦參加，但就是希望富士山賞臉，沒有被雲擋住。

若是選擇東京迪士尼作為人生第一個迪士尼的話，比較推薦先去陸地，因為陸地才有城堡，不過陸地就沒有賣達菲的商品。

這次是第一次遇到飛機延誤，直接延了8小時，如果是有保旅平險或是信用卡有附贈保險記得要跟航空公司索取「班機延誤證明」、飯店索取「住宿收據」、如果有用餐就記得留好發票，新航班的登機證也要留好。

💰 費用#

不含個人購物，共約 TWD 28000 / 人

1. 機票 TWD 11769 / 人
  • 樂桃：桃園→羽田：台幣5490 (無加購行李)
  • 捷星：成田→桃園：台幣6279 (無加購行李)
2. 住宿 TWD 5625 / 人
  • 因為是半夜抵達的班機所以住在機場附近的plat hostel keikyu haneda home
    雙人房一晚是：台幣1935，出羽田機場搭計程車過來大約10分鐘：台幣280(¥1300)，飯店半夜是採自助入住，衛浴是共用的，很乾淨，房間很小一間，比較推薦給半夜降落或是隔天一早要趕飛機的旅人
  • 後面三晚是住在Super Hotel 浜松町
    雙人房一晚：台幣3105，一樓櫃台有飲料吧，下午時段還會提供些許酒類，盥洗用品都可以在一樓拿取，這家飯店很酷的是可以選自己合適的枕頭，不過建議剛入住就趕快選喜歡的枕型，不然很可能被選光了。最後一天退房也可以將行李先寄放在飯店
3. 玩樂 TWD 4143 / 人
  • 富士山一日行程：台幣1705
  • 新宿御苑門票：台幣108
  • 迪士尼門票：台幣2330
4. 餐點 TWD 5000 / 人
  • 一份餐點大約：台幣300~600
  • 草莓一盒大約：台幣300~400，就已經很甜很好吃了
  • 迪士尼裡面的話餐點就貴一點
    雞腿：台幣130、漢堡套餐：台幣380、刈包：台幣130
5. 交通 TWD 1500 / 人

  • 其他幾天搭單趟大約23~39台幣，只有從市區搭去迪士尼武濱站是70台幣左右
