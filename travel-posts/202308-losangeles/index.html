去程：TPE → LAX
回程：LAX → TPE

🗓️ 行程#

Day1｜桃園機場 - 洛杉磯機場 - Hertz取車#

桃園機場 → 洛杉磯機場 → Hertz取車 → 🍽️ In & out → 🏠 Arcadia airbnb

Note

我們這次選Hertz，洛杉磯機場出口就會有接駁車的指示，只要找到你租車公司的Logo就可以上車，接駁車接送到取車處，到櫃檯填寫資料後至停車場取車，Hertz官網常常會有優惠碼，訂車的時候可以看一下有沒有適用的。接著回住宿的沿路上就先買了In & out漢堡當宵夜！

Day2｜農夫市集 - The Grove購物中心 - 比佛利山莊 - 郡立美術館 - 星光大道 - 格里菲斯天文台#

walmart → 農夫市集farmers market → The Grove購物中心 → 比佛利山莊+Rodeo Drive → 洛杉磯郡立美術館lalaland電影取景 →🍽️ Ktown吃SUN NONG DAN → 星光大道HollywoodWalkOfFame → 中國戲院ChineseTheatre → 格里菲斯天文台Griffith Park(賞夕陽/夜景) → 宵夜wendy burger → 🏠 Arcadia airbnb

Note

起床後先到附近的walmart採買，接著到farmers market去吃早午餐，然後到The Grove稍微逛一下後，就前往比佛利山莊+Rodeo Drive去朝聖了～接著到洛杉磯郡立美術館lalaland電影取景地標拍照，要注意這邊就會開始會有homeless出現！

下午到Ktown吃SUN NONG DAN韓式料理(如果吃膩漢堡可以來換換口味)。接著到星光大道HollywoodWalkOfFame這一區逛逛，最後到格里菲斯天文台Griffith Park看夕陽加夜景，那天剛好遇到土星沖日，所以人潮滿滿，停車位繞了好久才停在大約走路15分鐘處。

下山後就買了wendy burger當作宵夜，打算收集美國的所有漢堡名店～

Day3｜中央市場 - The last bookstore - 好萊塢湖公園 - 威尼斯運河區 - venice beach - Santa Monica#

中央市場GrandCentralMarket → 🍽️ eggslut → 天使鐵路AngelsFlight → The last bookstore → 🍽️ Tierra Mia Coffee → 開車逛DTLA → 迪士尼音樂廳 → 🍽️ cafe de mama → 好萊塢湖公園Lake Hollywood Park → 威尼斯運河區 → venice beach → SantaMonica → 🍽️ the albright → anaheim → walmart → 🏠 Kings Inn Anaheim

Note

中央市場GrandCentralMarket超推吃eggslut！真的很香很好吃！走路就可以到旁邊的天使鐵路，小小一段而已，但是體驗看看還不錯啦。接著也是從中央市場走路就可以到的酷書店The last bookstore。

因為DTLA治安比較差，所以我就只有開車逛逛，然後就前往可以拍到大大Hollywood標誌的公園Lake Hollywood Park拉，來這邊拍照加上曬曬太陽還不錯～威尼斯運河區這一塊蠻舒服的適合下車散散步。venice beach這邊就比較熱鬧會有許多人會在這邊運動、曬太陽等等。

接著傍晚到SantaMonica搭乘摩天輪，遊樂場有售票處，可以買單項設施，也可以買套票！雖然是夏天但是太陽下山後這邊蠻冷的，建議是要隨身帶個外套。在碼頭附近吃個晚餐就離開拉，這一區傍晚離開治安倒是沒有到令人害怕，但還是建議結伴同行。

因為明天要去迪士尼的緣故，我們就先前往anaheim去住宿拉！

Day4｜Disneyland#

Disneyland Park+Disney California Adventure Park 雙園 → 🏠 Kings Inn Anaheim

在購票完成後，立馬將門票認證連結至Disneyland的APP，確保該日入園預約成功。另外，有買Genie的話就可以抽幾項特定的遊樂設施，而且可以跨園預抽，如果一天要玩雙園區的話，比較建議加購才能玩到比較多設施。我們只有安排在Anaheim一天，所以玩雙園。

一大早7:30我們就準時先入園Adventure Park，我們的遊玩順序是：水鄉溫泉鎮賽車手Radiator Springs Racers → 星際異攻隊 Guardians of the Galaxy → 超特攻雲霄飛車 Incredicoaster → 玩具總動員瘋狂遊戲屋Toy Story Midway Mania → 蜘蛛人大冒險 WEB SLINGERS: A Spider-Man Adventure → 怪獸電力公司大眼仔與毛怪的救援行動 Monsters, Inc. Mike & Sulley to the Rescue → 入園Disneyland Park → 拍照城堡、吃午餐 → 冒險再續 Star Tours – The Adventures Continue → 巴斯光年星際探險 Buzz Lightyear Astro Blasters → 15:30 Magic Happens" Parade → 千年鷹號：走私者逃亡 Millennium Falcon: Smugglers Run → 米奇和米妮的飛速鐵路Mickey and Minnie’s Runaway Railway → 馬特洪峰雪橇飛車 Matterhorn Bobsleds → 巨雷山鐵路飛車 Big Thunder Mountain Railroad → 入園Adventure Park → 汽車總動員天地 Cars Land拍照參觀 → 入園Disneyland Park → 21:30 Wondrous Journeys with Fireworks → 入園Adventure Park → 22:15水舞World of Color → 入園 Disneyland Park → 印第安那瓊斯 Indiana Jones Adventure → 神鬼奇航之加勒比海盜Pirate of Carribbean → 23:55離園。

我覺得可以一日玩雙園，但是很耗體力08:00開園玩到24:00閉園，大家自己斟酌。而且要買快速通關！快速通關還能去排照相，會有電子檔照片可以下載。

Day5｜Ontario Mills - road trip：LA to Vegas#

Angel stadium → Ontario Mills mall → 🍽️ habit burger → road trip：LA to Vegas → baker溫度計 → Seven Magic Mountains 七魔山 → check in/停車 → Welcome to Fabulous Las Vegas Sign → 可口可樂商店 → m&m商店 → 🍽️ 手抓海鮮C3 seafood → 🏠 住宿Luxor

Note

這次剛好沒遇上球賽，但還是來Angel stadium參觀一下球場，接著去逛Ontario Mills mall，午餐買了habit burger後就開始我們的road trip拉！第一站先到barstow station休息站 → 接著路過baker world largest thermometer世界上曾經測到最高溫的溫度計打卡一下 → 然後到Seven Magic Mountains 七魔山打卡拍照，我們去的時候室外41度，真的要全身防曬加撐傘，不然下車一下感覺就快蒸發。

開著開著就結束4.5小時的沙漠路段之旅，先到今天的住宿金字塔Luxor酒店報到，然後再開車前往Las Vegas Sign拍照，這邊會有攝影師可以協助拍照，但是需要給小費。接著開車進行各式造型飯店巡禮後就去逛可口可樂商店和m&m商店拉。然後再開車到Cajun crab吃美式手抓海鮮～

Day6｜road trip：Las Vegas to 大峽谷南緣#

🍽️ White Castle → road trip：Las Vegas to 大峽谷南緣 → 胡佛水壩Hoover Dam → Kingman小鎮 → Seligman小鎮(汽車總動員) → Williams小鎮(加油便宜) → papillon grand canyon helicopter tours集合 → 搭直升機 → mather Campground information check in → Grand Canyon visitor center租帳篷 → mather Campground搭帳篷 → Yavapai Point看日落 → ⛺️ mather Campground

Note

今天買了White Castle當早餐後，再度展開下一趟road trip！第一站先到變形金剛的拍攝地胡佛水壩參觀，然後前往第一個小鎮Kingman下車伸展休息並跟Route66的招牌拍照。然後繼續前往下一個小鎮Seligman這邊有模仿汽車總動員的汽車裝飾可以拍照。剛好旁邊有一間小店snowcap賣Taco，可以買一份當點心吃～接著到Williams，這邊是稍繁榮一點的小鎮，加油也比較便宜，可以在這邊把油補滿再繼續出發！如果到這邊時肚子餓了，推薦可以吃cruiser’s route 66 cafe的美式烤肉！

接著抵達papillon grand canyon helicopter tours，這是我們在拉斯維加斯先報名好的直升機行程，報到後進行安全影片解說後，就搭上直升機前往大峽谷拉，整趟飛行大約30分鐘左右，會飛到大峽谷上方俯瞰整個大峽谷，非常震撼，很推薦這個活動！

下一站就是前往大峽谷拉，進入前每輛車要繳35鎂的車輛許可證費用。因為我們是出發前先在網路上訂帳篷營位，所以到mather Campground information報到領地圖及營位號，然後再前往Grand Canyon visitor center租帳篷，這邊是一間超市，除了租借露營用品，也可以購買食物等生活用品。回到營區位置將帳篷搭好後，就前往Yavapai Point等日落，太陽下山後，因為營區沒有光害，所以一抬頭就可以看到滿山滿谷的星空，超漂亮！

Day7｜大峽谷南緣 - 下羚羊峽谷 - 馬蹄灣 - road trip：馬蹄灣 to Vegas#

Mather Point看日出 → 大峽谷南緣 → 下羚羊峽谷 → 馬蹄灣Horseshoe Bend → road trip：馬蹄灣 to Vegas → Fremont Street → mirage幻景飯店火山秀 → 🍽️ White Castle → Bellagio百樂宮水舞秀 → 買Fat Tuesday → 飯店玩拉霸 → 🏠 Caesars Palace Hotel & Casino

Note

隔天一早5:30起床，驅車前往Mather Point等日出～接著還完帳篷後就趕快前往lower antelope canyon集合，這個是事先已經在klook上面報名好的行程，大家要注意從大峽谷過來的路上會經過不同時區，所以請在手機調好lower antelope canyon的時區時間，以提醒自己。到櫃檯報到後，就等領隊帶領大家出發，實際走樓梯下去後真的很開眼見，無法用言語形容的特別景觀，加上剛好有陽光灑進峽谷裡的話，拍照起來很漂亮！也可以請領隊幫忙拍照。

活動結束後，我們前往馬蹄灣Horseshoe Bend，從停車場走到觀景台大概要15分鐘，如果你是夏天去，一定要物理防曬加上撐傘，因為非常曬又熱！！！我們回到車上後馬上開到前方不遠處的麥當勞買可樂喝哈哈哈哈，接著就是再開4.5h回到拉斯維加斯拉～

因為回來已經天黑，所以我們就到Fremont Street參觀一下，這邊的人行道天花板是LED燈，也有很多喝酒的小店，所以這區晚上很熱鬧。接著就回到主街區去看mirage幻景飯店的火山秀和百樂宮水舞秀，然後回到飯店玩拉霸，但是不太會玩，都一直輸錢哈哈哈哈

Day8｜Las Vegas South Premium Outlets - road trip：Vegas to LA#

飯店游泳池 → 🍽️ Bacchanal(best buffet in vegas) → Las Vegas South Premium Outlets → road trip：Vegas to LA → 🏠 Ayres Hotel Laguna Woods

Note

今天訂了Vegas有名的buffet Bacchanal，所以早上就在飯店漂亮泳池玩水等吃飯，等午餐時間到櫃檯出示訂位資訊後，服務員會幫你帶位，實際品嚐下來，我覺得台灣的buffet還是贏很多XD

用完餐後先到Las Vegas South Premium Outlets稍微逛了一下，是半露天的，可以先上Premium Outlets官網查看各家品牌的折扣碼。然後就開車回LA拉～

laguna Beach → citadel outlet → Ontario Mills mall → 🍽️ habit burger → 🍽️ In & out → Hertz還車 → LAX搭飛機

Note

早上到laguna Beach散步後，就開車前往citadel outlet跟Ontario Mills mall購物拉！然後晚上到機場附近吃最後的In & out後就還車去機場了！

Day10｜洛杉磯機場 - 桃園機場#

總結#

Hertz租車、還車流程都還不錯，機場接駁車也很好認！到美國若有租車可以下載ParkMe Parking app，用來看附近停車場的停車費率。

直升機、Vegas、大峽谷、下羚羊谷、馬蹄灣這些都很推薦去，因為是真的是很特殊的景觀。

體力夠的話Disney請玩雙園區Disneyland Park+Disney California Adventure Park，而且要買快速通關！

美國刷卡蠻普遍的，主要花現金都用在加油上，現金會稍微便宜一些加上比較不容易被盜刷。現金加油就是去櫃檯告訴工作人員油槍號碼，然後你要加多少錢，這時候就可以走回去加油了，油槍到你加的金額或加滿就會跳停了，如果有餘額再去櫃檯領回。

不論是在路邊或是加油站等其他人上車的時候，車內的人都把車上鎖會比較安全，這次就有遇到流浪漢在加油站內閒晃的情況。
