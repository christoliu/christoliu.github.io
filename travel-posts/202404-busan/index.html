🇰🇷 釜山

1 min

Time：2024/04#

✈️ 航班與入境#

去程：TPE → PUS
回程：PUS → TPE
此次旅程時韓國免簽，只需3天內上網填寫【健康狀態調查表】

🗓️ 行程#

Day1｜桃園機場 - 釜山機場 - 南浦商圈#

桃園機場 → 釜山機場 → 🍽️ 南浦商圈晚餐 → 🏠 Staywell Hotel

Note

傍晚抵達釜山機場後，出口走大約5分鐘就可以搭乘機場輕軌，買票後搭乘至sasang轉地鐵到seomyeon再轉到Nampo，出口走約2分鐘就是我們的飯店，check in後就到南浦商圈吃晚餐，結束第一天。

Day2｜skyline Luge - 海東龍宮寺 - 膠囊列車 - 釜山塔#

🍽️ 早餐EGGDROP → skyline Luge → 海東龍宮寺 → 🍽️ 午餐 → 膠囊列車 → 🍽️ 晚餐道熙家 → 釜山塔 → 🍽️ 宵夜橋村炸雞 → 🏠 Staywell Hotel

Note

早餐在南浦商圈點了EGGDROP，蠻多人的要等，所以我們就先到附近的 BUSAN Tourist Hotel領在klook上訂好的BUSAN Pass Big3實體版，現場出示網路預訂的QR就可以兌換了，是一張類似悠遊卡的卡片，商店跟地鐵站都可以儲值拿來搭交通運輸，然後Big3版本是可以選1個紫區景點、2個藍區景點(Klook上面有列出)。紫區我們是選玩skyline Luge，所以吃完早餐(吐司煎得很香蠻推薦的)，領完BUSAN Pass之後就接著搭公車去玩skyline Luge了。抵達後先到櫃檯用BUSAN Pass領取票券，接著選安全帽後到排隊區域搭纜車(有點像公園的條狀椅子加前面有一條保護的橫桿)，吹風等待抵達滑車出發點蠻舒服的。工作人員會用教大家控制滑車，有三種模式：啟動、煞車、完全停下，蠻好控制的，大人小孩都可以玩。

之後就到對面的海東龍宮寺，稍微走一段上坡後會先遇到一個賣攤販的區域，再往內走就是海東龍宮寺的入口，進去不用門票，蠻值得進去看看。逛完之後回到攤販區吃小吃當作午餐，接著前往天空膠囊列車尾浦站。

出發前有先在網站上看購票情況，每個時段都預約滿了(建議確定要去的人越早預約越好)，只好到現場碰碰運氣，很顯然運氣是不夠好QQ，最好只買了單程的海岸列車尾浦 → 青沙浦的單程票。接著沿著步道散步至青沙浦天空步道後，再折返回道熙家吃烤扇貝當作晚餐。

接著我們搭計程車至附近的公車站，大約搭乘1小時後抵達到釜山塔(BUSAN Pass的藍區景點1)，上去看夜景還有互動式的拍照機，蠻有趣的。下塔後就到附近買橋村炸雞當宵夜，然後回飯店休息拉～～

Day3｜松島纜車 - 廣安里海邊 - 白淺灘文化村 - 南浦洞商圈 - 釜山機場 - 桃園機場#

🍽️ 早餐 → 松島纜車 → 廣安里海邊 → 白淺灘文化村 → 南浦洞商圈 → 釜山機場 → 桃園機場

Note

早上買了杯咖啡後，悠閒的出發前往松島纜車站，我們是搭乘到山上的站為起始站，抵達後拿著BUSAN Pass到櫃檯就可以換來回的纜車乘車券(BUSAN Pass的藍區景點2)，因為是平日沒有太多人，大約排隊5分鐘就搭上纜車了，整個纜車是跨海的，坐著欣賞風景蠻Chii的～抵達廣安里站之後，我們出纜車站到廣安里海灘散散步，接著搭回程纜車～

下一站是搭公車前往白淺灘文化村，下公車後穿越房子中間的小巷子就可以到一長條人行道，往前看就是大海，步道上面有很多紀念品手工小店以及咖啡。

最後回到南浦洞商圈吃韓式烤肉店後，就在南浦洞購物，然後搭車去機場拉，我們回程搭的是濟州航空，是在機場的自助報到機報到以及免費選位置。機場沒有什麼可逛區，只有兩家超商，過海關之後裡面還有超商跟幾家伴手禮店。

總結#

四月的天氣蠻舒服的，短袖加一件薄外套即可。另外，釜山的交通蠻方便的，機場出來就可以搭輕軌轉地鐵，景點之間也是用google導航要去的景點後，跟隨導航指示搭車就可以抵達(跟在台北搭公車差不多的感覺)，大部分店家都可以刷卡，只有小吃攤需要付現金。釜山景點也算蠻多的，排滿五天沒問題(還可以安排汗蒸幕、樂天世界遊樂園等等)，想要休閒一點的旅人，釜山也有蠻多看海又舒適的咖啡廳可以悠閒的渡過下午～

  • 濟州航空：釜山→桃園：台幣3327
2. 住宿 TWD 735 / 人

3. 玩樂 TWD 1064 / 人
  • BUSAN Pass(Big 3實體版)

5. 交通 TWD 350 / 人
  • 我們一抵達機場就開啟機場輕軌、地鐵、公車的大眾運輸移動方式
