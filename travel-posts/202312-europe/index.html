✈️ 航班與入境#

去程：TPE → DXB → MUC
回程：MUC → DXB → TPE
此次旅程歐洲申根國免簽

🗓️ 行程#

Day1｜桃園機場 - 杜拜機場轉機 - 慕尼黑機場 - 慕尼黑市區 - 瑪利亞廣場慕尼黑聖誕市集#

桃園機場 → 杜拜轉機 → 抵達慕尼黑機場 → 搭乘漢莎巴士到München Hauptbahnhof → 住宿(朋友家)放行李 → 瑪利亞廣場慕尼黑聖誕市集Münchner Christkindlmarkt → 🍽️ 鮭魚堡 → 🏠 朋友家

Note

在桃園機場時就先拿到兩段機票，下飛機前也會提前將轉機的資訊顯示在螢幕上，抵達杜拜機場後就跟著非常清楚的轉機指示，就能抵達下一段飛機的登機口。雖然是半夜抵達，但杜拜機場的免稅店仍然都是開著的，不怕沒地方逛！

抵達慕尼黑機場後，會有漢莎巴士的指引，跟著指引就能到搭乘處。我是先在漢莎巴士的官網購買好來回的巴士票，上車後出示車票的QR碼給司機掃描即可，巴士下層也有自行放置大型行李的行李區，沿路有幾個下車站，我是在終點站慕尼黑中央車站下車。

到朋友家放完行李後，第一站當然先逛聖誕市集!! 所以我們就到瑪利亞廣場的Münchner Christkindlmarkt感受一下歐洲的聖誕市集氛圍，順便欣賞新市政廳。一抵達市集就去吃鮭魚堡跟必喝的熱紅酒!!!據說每個聖誕市集店家都有自己特色的熱紅酒馬克杯，因為買紅酒是會先付杯子的押金，若是有喜歡的杯子，就可以直接帶走，不用回店家換回押金～

Day2 🇩🇪｜慕尼黑工業大學 - Tollwood聖誕市集 - 酒吧lost weekend munich#

慕尼黑工業大學 → Tollwood聖誕市集 → 🍽️ 酒吧lost weekend munich → 🏠 朋友家

Note

今天起床後陪朋友到慕尼黑工業大學聽課，順便玩校園裡的超長溜滑梯!!!是誰想到要在大學裡面建溜滑梯的拉，但是好好玩ㄟ!!!之後又去Tollwood聖誕市集，這個市集比較小，不過有現場樂團表演，可以邊聽邊喝酒，很Chill。後來我們又去lost weekend munich酒吧續攤!!

Day3 🇩🇪｜寧芬堡 - BMW WELT - 奧林匹克塔#

寧芬堡schloss nymphenburg → 🍽️ 午餐haxengrill德國豬腳 → BMW WELT → 奧林匹克塔Olympiaturm看夜景 → 🍽️ 晚餐89anju → 🏠 朋友家

Note

早上起來天氣不太好，不過我們還是去參觀了漂亮的寧芬堡，還去吃了德國豬腳和啤酒，這邊的豬腳配菜是一個很像發粿的馬鈴薯球(吃起來蠻特別的哈哈哈哈)。接著跑去參觀BMW展示中心，是不用門票的，裡面有一些可愛的小車讓大家參觀！接著我們上去奧林匹克塔看夜景，入口處就可以買門票了！晚餐去吃韓餐89anju(如果已經吃膩歐洲食物，可以來換換口味)。

Day4 🇩🇪🇨🇿｜前往布拉格 - 舊城廣場聖誕市集 - 天文鐘 - 查理大橋#

Munich central bus station → 搭Flixbus → Prague Main Railway Station → 布拉格舊城廣場聖誕市集 → 天文鐘 → 查理大橋 → 晚餐u parlamentu → 🏠 Rezidence Štepanska

Note

一早到巴士站搭Flixbus前往布拉格，這段路我買到的票是自由選位置的，座位上也有附USB充電器、廁所。大約5小時後抵達布拉格拉！先到airbnb放行李休息一下後，就去逛布拉格的聖誕市集拉!!!路上還會遇到馬車載人從面前經過～後續又去了有名的天文鐘跟查理大橋逛逛。晚餐去吃u parlamentu，會先送上一盤迎賓餅乾，我點的主餐是烤鴨，蠻好吃的，也不算貴，推薦大家可以去吃看看。

Day5 🇨🇿｜跳舞的房子 - cafe louvre - 布拉格城堡 - 聖維特主教堂 - 聖喬治教堂 - 舊城廣場聖誕市集#

跳舞的房子Tančící dům → 🍽️ 早午餐cafe louvre → 布拉格城堡 → 聖維特主教堂Katedrála svatého Víta → 聖喬治教堂Bazilika svatého Jiří → 🍽️ 布拉格舊城廣場聖誕市集 → 🏠 Rezidence Štepanska

Note

因為時差一早就醒來了，跟同伴先到跳舞的房子踩點打卡，接著散步到cafe louvre吃早午餐，這家是是歷史悠久的咖啡館，卡夫卡、愛因斯坦等都是這裡的常客。(不過我覺得來喝咖啡跟吃點心就可以了)。接著我們去布拉格城堡參觀，整區蠻大的，應該可以逛3小時，特別有興趣的話，還可以買票入城堡內參觀，待到晚上的話，可以到旁邊的星巴克喝咖啡、看夜景。

Day6 🇨🇿｜庫倫洛夫Český Krumlov（CK小鎮）#

🍽️ 早餐paul可頌+拿鐵 → 布拉格巴士站Praha Na Knížecí → 搭巴士RegioJet約3小時 → 庫倫洛夫(ck小鎮)巴士站Český Krumlov, AN → 🍽️ 晚餐魚排 → 搭巴士RegioJet約3小時回布拉格 → 🍽️ 宵夜麥當勞 → 🏠 Rezidence Štepanska

Note

早餐沿路買了paul可頌+拿鐵，可頌真是好吃，接著我們到布拉格巴士站Praha Na Knížecí搭乘巴士RegioJet約3小時抵達庫倫洛夫(ck小鎮)！，不過今天天氣不太好，有看過人家分享大晴天或是下過雪會更漂亮！找到一家svejk restaurant吃晚餐(魚排還不錯吃)，接著搭巴士回布拉格後去買麥當勞當宵夜吃，還遇到聖誕裝扮的布拉格電車！

Day7 🇨🇿🇭🇺｜市集havelske trziste - Obecní Dům 慕夏的壁畫導覽 - palladium - 前往布達佩斯#

郵局寄明信片 → 早午餐venue → 市集havelske trziste → Obecní Dům 慕夏的壁畫導覽 → 逛百貨palladium → Prague Main Railway Station → 搭Flixbus夜車

Note

一早先到郵局寄明信片，一張郵票要70台幣(好貴)！接著去吃早午餐venue，它的鬆餅炸雞好好吃！！旁邊剛好有一個市集可以逛逛，天文鐘紀念磁鐵可以在這邊買。接著到火藥塔旁邊的市民會館Obecní dům參加慕夏的壁畫導覽(入口可以購票，有場次時間表)。到百貨palladium逛逛吃沒美食街後，就到布拉格火車站等待要去布達佩斯的Flixbus夜車(要搭6小時)，因為會跨國家，所以上車前，司機會先簡單檢查一下護照。

Day8 🇭🇺｜漁人堡 - 弗洛斯馬提廣場聖誕市集 - fashion street - 遊船夜遊多瑙河#

Budapest Népliget站 → 🍽️ 早午餐New York cafe → 漁人堡 → 弗洛斯馬提廣場聖誕市集 → fashion street → 遊船夜遊多瑙河 → 🍽️ 晚餐menza → 🏠 FREEDOMFIGHTER DESIGN FLAT 1

Note

一大清晨抵達布達佩斯拉！就先到早午餐New York cafe等八點開門，不用排隊可以直接入內用餐，大約九點外面就開始大排長龍(建議大家是一開門就去)，裡面裝潢真的很漂亮也可以拍照，但是餐點就普通且偏貴(真的就是去吃氣氛的)。

接著我們搭車去漁人堡，這邊的風景真的很漂亮!!!推薦必去!!!接著就在市區逛逛，天氣好的話到處都很美！接著傍晚先到聖誕市集簡單吃晚餐後，到fashion street逛逛，整條街都很有聖誕氣氛。

我們是搭晚上時段的多瑙河遊船，記得要先提早線上購票，購票時會依照不同的船隻公司給不同的碼頭編號，總航程大約1小時，這行程也很推薦，必去！

結束之後到menza吃晚餐！這間超好吃，推薦鴨胸燉飯和匈牙利燉牛肉湯(Goulash)!!

Day9 🇭🇺｜中央市場 - 自由橋 - 國會大廈 - 廢墟酒吧#

🍽️ 早午餐mosselen → 中央市場 → 自由橋liberty bridge → 國會大廈散步 → 🍽️ Drum Cafe平價匈牙利餐廳 → 🍽️ Szimpla廢墟酒吧 → 🏠 FREEDOMFIGHTER DESIGN FLAT 1

Note

早餐去吃mosselen，柳橙汁很好喝。接著去逛中央市場，買了便宜又好吃的綠葡萄。ALDI超市的麵包很便宜又好吃，如果旅費已經超標的人，可以考慮這個！可以到自由橋拍照，不過車流還是蠻大的，要注意安全。

Drum Cafe是平價的匈牙利餐廳，CP值很高，蠻推薦的！Szimpla廢墟酒吧也很好玩，酒的價格跟台灣差不多，買完酒之後可以自己找空位坐下來聊天，但是裡面人很多蠻擠的，移動要注意安全。

Day10 🇭🇺｜聖伊什特萬聖殿 - 英雄廣場 - 沃伊達奇城堡 - indoor minigolf#

聖伊什特萬聖殿Szent István-bazilika → 英雄廣場 → 沃伊達奇城堡vajdahuntad vara → 國會大廈散步 → 🍽️ 晚餐elso pestireteshaz → indoor minigolf → 🏠 Wombat’s City Hostel Budapest

Note

今天天氣很好晴空萬里，早上先到聖伊什特萬聖殿廣場的市集覓食，接著散步去英雄廣場跟沃伊達奇城堡逛逛曬太陽。

晚餐吃elso pestireteshaz，匈牙利燉牛肉湯是匈牙利的特色，基本上每家餐廳都會有且都很好吃，而且湯裡面有很多料還會附上麵包，所以CP值很高！

indoor minigolf入場是3500福林(大約台幣315元)，會幫你把外套掛起來，給你高爾夫球竿、球、計分表，接著入場就可以自己挑關卡挑戰，每個關卡不限時間，所以有時候要排隊一下，在這裡面大概可以玩3小時左右，大廳也有調酒/飲料/點心可以購買，很推薦來玩!!!!

Day11 🇭🇺🇦🇹｜前往維也納 - 內城區 - 聖史蒂芬大教堂 - ankeruhr時鐘 - 市政廳公園 - 聖史蒂芬大教堂#

搭Flixbus → 奧地利維也納 → 🍽️ 香腸 → 內城區Innere Stadt → 聖史蒂芬大教堂Stephansdom → ankeruhr時鐘 → 🍽️ 維也納薩赫咖啡館Cafe sacher → 市政廳公園 → 聖史蒂芬大教堂等跨年 → 🏠 a&o Wien Hauptbahnhof

Note

一早搭Flixbus去維也納，車程大約3.5小時。先去買維也納地鐵二日券，然後到住宿的地方寄放行李後，前往innere stadt逛逛，順路到附近的ankeruhr時鐘欣賞一下，整點會報時，但沒有很特別哈哈哈哈，可以路過就好，不用特別過去看。接著去有名的Cafe sacher吃巧克力蛋糕，但我覺得蠻普通，如果嚐鮮可以去拉！路上經過小餐車有賣leberkäse有點像烤好的麵包夾午餐肉，蠻好吃的，推薦可以吃看看。

維也納跨年有蠻多區，我們先搭地鐵到市政廳公園，市政廳公園這區有人在舞台上演唱，但整體蠻小的，所以後來我們又回到聖史蒂芬大教堂正前方的舞台等跨年，到數跨年後教堂會敲鐘伴隨著藍色多瑙河圓舞曲，周圍得人會開始跳華爾滋，蠻酷的體驗！

Day12 🇦🇹｜美泉宮 - 美景宮#

🍽️ 早餐沙威瑪 → Schloss Schönbrunn熊布朗宮(美泉宮) → musikverein wien金色大廳音劇沒買到票 → schloss belvedere美景宮 → 🍽️ 晚餐炸豬排centimeter → 🏠 a&o Wien Hauptbahnhof

Note

隔天早餐先吃附近的沙威瑪後，搭車去美泉宮參觀，買票入內參觀會附上導覽器，走到每一區導覽器就會解說相關的歷史故事。室內結束之後可以到戶外往山上拱門的方向走，爬上去後往下看很漂亮，這邊整個就很悠閒又放鬆!!!

喜歡音樂者也可到musikverein wien金色大廳聽演奏，不過我們這次剛好沒票了ＱＱ接著我們到美景宮戶外參觀，建築真的很漂亮呢。

來維也納推薦去吃炸豬排，所以晚餐我們去吃centimeter，這家也是CP值很高，餐點好吃又不貴，不過服務生不太會理你就是了，但也不影響用餐拉XD

Day13 🇦🇹🇩🇪｜薩爾茲堡 - 米拉貝爾宮 - 格特萊德街 - 莫札特的家 - 薩爾斯堡馬槽 - Kapitelplatz廣場#

火車Wien Westbahnhof-Salzburg → 米拉貝爾宮Schloss Mirabell → 格特萊德街Getreidegasse → 莫札特的家mozarts geburtshaus → 薩爾斯堡馬槽pferdeschwemme → Kapitelplatz廣場大金球 → 🍽️ 晚餐炸雞barenwirt → 火車Salzburg-München Hauptbahnhof→ 🏠 朋友家

Note

回程我們先搭火車到薩爾茲堡，參觀了米拉貝爾宮以及莫札特的家和Kapitelplatz廣場的大金球，悠哉地度過一個下午。搭車之前我們走到barenwirt去外帶炸雞，這家炸雞很好吃!!!如果時間充裕的話可以內用應該會更美味。接著就繼續搭乘火車回到慕尼黑拉～

慕尼黑中央火車站的麥當勞有賣捲捲薯條，搭車前後可以去買來吃，吃起來比一般薯條鹹一點，整體還不錯。

Day14 🇩🇪｜慕尼黑機場 - 杜拜轉機 - 桃園機場#

München Hauptbahnhof搭漢莎巴士去機場 → 慕尼黑飛機 → 杜拜轉機 → 抵達桃園機場

Note

在慕尼黑中央車站搭漢莎巴士去機場，這是來之前就先買好的來回巴士票，一樣上車出示QR碼給司機掃描就可以上車了，如果帶很多行李的話，蠻推薦這種方式來回機場。

在慕尼黑機場報到櫃檯就可以拿到兩段機票，行李也可以直掛，託運完行李後可以到附近逛逛或是去退稅，但是因為這次我沒有買到可以退稅，所以就沒有體驗到退稅流程，大家可能要再上網爬文一下。

抵達杜拜，位置螢幕上一樣會顯示轉機登機門的位置，下飛機後過安檢，就可以去買伴手禮拉～推薦可以買阿拉伯人造型罐子的花生以及椰棗，是杜拜機場的特色伴手禮。

總結#

中歐真的相對安全，花費也相對較低，吃餐廳一次大概350-500台幣，不過建築真的很類似，到後面有點沒驚喜感。另外，德國建議把國王湖、新天鵝堡、哈修塔特排進去，捷克/匈牙利/維也納，我覺得包含交通日，各三天也就都足夠了。

小資族的話蠻推薦搭Flixbus夜巴，省一晚住宿、又可以通勤、又便宜。人多的話蠻推薦找airbnb平分下來住宿費蠻省的。

季節的話，蠻推薦聖誕節前後去，一方面可以體驗歐洲的聖誕氛圍，一方面貼身包背在最內層不熱且外面還有很多層外衣、外套可以保護貼身包！
