🇨🇳 上海

1 min

Time：2024/10#

✈️ 航班與入境#

去程：TPE → HKG → PVG
回程：PVG → TPE
此次旅程時使用台胞證入境

🗓️ 行程#

Day1｜桃園機場 - 香港機場 - 上海浦東機場#

桃園機場 → 香港轉機 → 抵達上海浦東 → 🏠浦東亞朵飯店

Note

因為太晚買機票，直飛的都超級貴，只好選要轉機的。降落後撥打電話請飯店來機場接駁，check in後就下載美團APP訂宵夜到飯店享用，結束轉機奔波的一天。

Day2｜迪士尼#

🍽️ 早餐飯店 → 上海迪士尼 → 🍽️ 宵夜胖子燒烤 → 🏠浦東亞朵飯店

Note

大約8:30才到迪士尼園區，沒有排到隊，很順的跟著人潮到驗票口出示提先預訂的門票QR code，掃描後人員會發紙本門票，入園後用上海迪士尼APP掃描紙本上的QR將門票綁定至APP。當天去的人流不多所以開放抽預約等候卡的只有米奇妙游童話書秀。

我們的遊玩順序是：先衝去動物方程式園區熱力追蹤 → 園區內散步、拍照、吃爪爪冰(仙楂味)。 → 七個小矮人礦山車，排單人通道大概也排了30分鐘 → 加勒比海海盜，排大概10分鐘 → 買午餐泡菜牛肉捲跟萬聖節雞肉捲餅去城堡前面的椅子吃 → 觀看13:00金色童話盛典秀 → 14:00-14:30米奇妙游童話書秀 → 15:00-15:15米奇童話專列遊行 → 逛大街上的商店，才發現萬聖節系列的達菲商店需要有中籤(一張門票一天可以抽一次)才能進去逛跟購買，只好空手離開 → 明日世界園區玩創極速光輪，排隊30分鐘(乘坐設施是趴著，第一段加速有點快，接下來就有點像太空山的感覺！) → 巴斯光年星際營，大概只排了15分鐘 → 又去玩一次創極速光輪，排隊20分鐘 → 回到動物方程式園區玩熱力追蹤，大概排了20分鐘 → 18:30唐老鴨萬聖夜反派遊行 → 回到動物方程式園區拍晚上的街景 → 20:15的奇夢之光幻影秀。

煙火結束後到停車場等飯店的接駁車，回飯店放完東西後，馬上叫車去大桶大足浴按摩店，放鬆一下在迪士尼奔波整天的疲勞，足浴按摩店特色是，自己一個包廂，可以邊泡腳、按摩邊看自己挑選的電影來看，而且費用也不高(80分鐘 / CNY192)。

接著到飯店對面的嚕串店，點了串燒跟麻辣湯，最酷的品項大概是玉米，玉米是一粒一粒被串起來烤!!

Day3｜宮宴 - 烏鎮#

🍽️ 飯店早餐 → 靜安寺站 → 🍽️ 午餐宮宴 → 上海南站 → 烏鎮 → 🏠烏鎮民宿

Note

在飯店享用完早餐後，搭了大概40分鐘的地鐵到靜安寺站，步行大約10分鐘後，就可以抵達宮宴這家餐廳(這家餐廳是打造成古裝的用餐風格及表演，需要先在微信上面訂位置)，到櫃檯報到時，櫃檯人員會詢問你是否需要加購換古裝及妝髮的服務，我們有加購，因此會拿到兩個牌子，並且指引我們到座位上等待，輪到我們換好衣服、化好妝之後就可以到庭園和座位上拍照，等待12:10準時開席。整體下來，我覺得蠻酷的，有種電視上才會看到的感覺!!!

用餐完畢後，我們搭地鐵到上海南站，搭乘前往烏鎮景區的大巴去程大約1h40m就抵達烏鎮西柵景區門口，進去櫃檯check in後，馬上到旁邊的搖擼船排隊(夕陽下山前30分鐘蠻適合搭乘的，可以同時看到白天跟晚上的風景)，接著就是一連串的吃吃吃吃吃啦！烏梅青號飲料、永平粉團鋪、書生羊肉面、錦記糕點鋪、醬鴨店、糖潮甜品。吃飽喝足後到烏鎮郵局去寄明信片。

Day4｜烏鎮 - 靜安寺#

🍽️ 早茶客 → 上海南站 → 靜安寺站 → 🍽️ 萊萊小籠 → 🏠 浦東亞朵飯店

Note

7:30我們就起床先到早茶客享用早點：桥里桥烧饼铺、生煎包、桂花糕，這邊是一個早上的市集，可以到各店鋪去購買自己喜歡的早餐，然後到用餐區找座位食用。有些酒店或是民宿有跟早茶客配合，可以將早餐換成早茶客的抵扣金，需要一到景區報到時就先跟櫃台登記，若是名額已經滿了，像我們一樣自掏腰包來享用早餐也是可以的。

吃完早餐後，逛了一下喜慶堂，買到兩個漂亮的烏鎮磁鐵，然後因為我們是在抖音上面購買的套票，有包含草本染坊的染布體驗，所以就到染坊去體驗染布啦，整個過程大概30分鐘可以完成。染完後就去吃茅老太臭豆腐，只能內用，不過如果你又自備提袋，就可以外帶！

回程也是在西柵景區門口搭乘大巴，回程比較塞車，大該約要2h30m才會抵達上海市區，抵達後剛好有碰到瑞幸咖啡，就買了有名的醬香拿鐵(茅台+拿鐵)來嚐嚐，個人覺得喝起來有點奶酒味，接著搭地鐵到靜安寺，走路去吃萊萊小籠，蟹黃麵跟小籠包螃蟹的味道都蠻重的，推薦大家可以來吃！

太陽下山後，我們逛到靜安寺附近的一座橋上等靜安寺點燈，然後到附近的百貨商場裡面買有霸王茶姬飲料，這次點了伯牙絕玄，喜歡喝奶茶的應該都會喜歡!而且他們的吸管是三細管成一支吸管，據說每管會喝到不同的味道，但我是喝不出來拉。

本來要去吃鴻姊麻辣火鍋，但要現場排隊快2h，所以我們就放棄排隊，回到住宿附近的大桶大足浴店又進行了一次按摩後，回飯店用美團點了麻辣乾鍋跟串串吃～

Day5｜浦東機場 - 桃園機場#

🍽️ 早餐飯店 → 浦東機場 → 桃園機場

Note

因為是一早的飛機，所以我們搭乘飯店提供最早一班的機場接駁車到浦東機場T2，大概30分鐘就完成櫃檯托運+安檢+過海關，到登機口的路上發現了藍瓶咖啡，就順手買了一杯拿鐵來喝，然後登機口門口也有星巴克。

總結#

這次是10月中去，天氣已經轉涼很舒服，加上中國剛放完十一長假，因此迪士尼跟烏鎮的人潮都算蠻少，個人覺得這個時間點去上海超級合適！建議大家早點買機票，費用會比較便宜，直飛也比較省時間。

浦東機場離市區比較遠，但是離迪士尼近，而且大部分住宿會提供迪士尼與機場接駁車，所以如果主要是去玩迪士尼的人可以選擇浦東進虹橋出，或是反過來。

宮宴這家餐廳在微信小程序上可以訂位並且選位子，推薦坐在中間一點，盡量避開柱子，否則有些表演會被擋到。

烏鎮晚上白天都蠻漂亮，建議至少住一個晚上。
