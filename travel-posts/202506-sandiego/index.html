🇺🇸 聖地牙哥

3 min

Time：2025/06#

✈️ 航班與入境#

去程：TPE → SFO，SFO → SAN
回程：ONT → TPE
此次旅程使用之前申辦的ESTA入境(期效3年)

🗓️ 行程#

Day1｜桃園機場 - 舊金山機場#

桃園機場出發 → 抵達舊金山機場 → 🍽️ In & out → 🏠 親戚家

Note

如果主要是去聖地牙哥，其實可以飛到洛杉磯租車，然後洛杉磯和聖地牙哥一起玩！

Day2｜舊金山機場 - 聖地牙哥機場 - La Jolla Cove - UCSD#

舊金山機場 → 聖地牙哥機場 → 🍽️ Oscar’s mexcian Taco → La Jolla Cove → UCSD → 🍽️ Mitchell seafood → 🍽️ Somisomi → 🏠 Hyatt Regency La Jolla at Aventine

Note

搭一大早的飛機從舊金山出發前往聖地牙哥，雖然是國內線，但是排隊安檢的人還是很多，建議大家還是預留2小時到機場。首次搭聯合航空，整體感受都還不錯！

抵達後，我們先到Oscar’s mexcian Taco吃午餐，這家的海鮮很新鮮，配Taco也很搭！La Jolla Cove就是看海獅的地方，海獅真的超多，但有點海邊的臭味就是了。下午到UCSD去參觀校園，買杯咖啡邊散步邊逛，還不錯。晚餐去吃Mitchell seafood，這家的海鮮很新鮮也處理得很好吃，如果天氣好大太陽的話，記得選戶外座位可以邊看夕陽邊享用餐點。Somisomi是鯛魚燒冰淇淋當飯後甜點還不錯！

Day3｜Coronado Island - imperial beach - US/Mexico Border - little italy - waterfront park - Balboa Park#

Coronado Island → imperial beach → US/Mexico Border → omomo → little italy → waterfront park → 🍽️ Cucina urbana → Balboa Park → 🍽️ taco el gordo → 🏠 Hyatt Regency La Jolla at Aventine

Note

從聖地牙哥開車經過跨海大橋進入Coronado Island，島上的Hotel Del Coronado，大家可以去參觀！Coronado Ferry Landing旁邊可以拍聖地牙哥天際線，加上這區有一些小店、餐廳，可以沿著海邊步道散步。imperial beach天氣好的話，可以散散步。原本US/Mexico Border有一個海灘，但是現在不開放了，所以我們只有沿路看到山上的邊界牆。little italy有很多義大利餐廳，也有很多人在路邊洋傘下喝咖啡曬太陽。Cucina urbana是很好吃的義大利餐廳，建議先訂位！Balboa Park下午來逛逛散步也很不錯。taco el gordo的仙人掌口味很好吃，推薦!!!

Day4｜Old Town - USS Midway museum - embracing peace statue - Ruocco park - Petco Park#

Old Town → 🍽️ izola bakery → USS Midway museum → embracing peace statue → Ruocco park → Petco Park → 🏠 Hyatt Regency La Jolla at Aventine

Note

Old Town建議週末來，平日有些參考的地方會沒開放，整區有很多墨西哥風格的小店可以逛，也有餐廳可以用餐，走走逛逛還不錯。izola bakery可以先線上訂後時間到去取餐，或是快關門的時候去碰碰運氣，我們關門前去買沒排到隊，但剩下原味可頌可以買，他們的可頌外層是酥脆的，蠻好吃！不過店附近接到感覺是比較髒亂的區域，雖然沒遇到太多奇怪的人，不過街道蠻臭的！

USS Midway museum可以現場買票，買票後進入航母，可以領導覽器，右邊有一個小播放室，一天有幾場可以看，要認真每層都逛完大概要一整個下午，劍橋參觀解說最晚只到4點，如果第一天太晚去沒逛夠或很熱愛這些的人，離開前可以到票務櫃檯請他幫你蓋章領隔天的免費入場券，但要注意隔天入場時會核對證件確保是本人！

參觀完畢沿路散步走走停停到Petco Park看棒球！要注意美國看棒球有規定的包包尺寸，若東西比較多建議是帶全透明提袋，出發前記得先查一下各球場包包尺寸規定！我們住比較遠然後搭trolley過來看球，球賽結束時會有很多球迷也都搭trolley所以安全上沒有什麼疑慮，除了開車到球場，大家也可以考慮這個交通方式(畢竟球場停車真的貴桑桑)。

Day5｜The gun range - USS Midway museum - Seaport Village - Petco Park#

The gun range → 🍽️ taco el gordo → USS Midway museum → Seaport Village → Petco Park → 🏠 Hyatt Regency La Jolla at Aventine

Note

吃完早餐後搭車到The gun range打靶，初次在美國體驗真槍射擊場，店員人很好會推薦適合你的型號，進場後也有工作人員手把手教學，不過靶場裡面聲音真的蠻大的，所以我們體驗完就趕快跑出來了。

搭公車回到市區又去吃一次taco el gordo！然後搭巴士去昨天來不及參觀的USS Midway museum劍橋解說。結束之後沿路走到Seaport Village逛逛，沿海有一些椅子跟陽傘可以坐下休息看海發呆。然後就又去Petco Park看棒球拉！

今天主要都搭大眾運輸，要下載APP：PRONTO，註冊後綁卡儲存虛擬交通卡，儲存最少金額是USD3，所以大家可以算一下會搭幾趟，再做儲值。Trolley + Bus一天最高上限付款是USD6，單趟USD2.5，每次刷完2小時內可以繼續免費搭乘。

Day6｜Desert Hills Premium Outlets - oshua Tree National Park#

Desert Hills Premium Outlets → Joshua Tree National Park看夕陽 → Walmart採買 → 🏠 Joshua Tree zen Airbnb

Note

一早就租車離開聖地牙哥，前往Desert Hills Premium Outlets，這個Outlet是全露天的，但是每個店家外的走廊是有遮陽的，所以不會曝曬，這裡是一個比較多精品的Outlet，如果有興趣的話，推薦中停這邊休息加上購物！

之後就前往Joshua Tree看夕陽，個人覺得夏天的話這邊適合下午五點之後再來，不然真的太熱太曬了！等太陽微下山後，風就是涼的，然後看夕陽很美！

大概離住宿5-10分鐘車程就有Walmart，因為這次住Airbnb廚房設備很齊全，所以我們就到Walmart去買了一些牛排、沙拉、義大利麵、麵包、蛋、飲料等等，準備這幾天的餐食！

Day7｜Joshua Tree National Park - VillageFest#

Joshua Tree National Park → VillageFest → 🏠 Joshua Tree zen Airbnb

Note

因為白天真的太曝曬太熱了，所以我就在Airbnb裡面看影片、玩桌遊，等下午不那麼熱的時候再出去玩Mini Golf(但還是要全身防曬)！等到下午5點多我們再到Joshua Tree National Park裡面幾個打卡點走走逛逛。週四剛好Palm Springs大街上會有小市集活動，所以就安排到這邊走走逛逛！

Day8｜Palm Springs Aerial Tramway - Dodger stadium#

Palm Springs Aerial Tramway → Palm Canyon Drive → 🍽️ IN N OUT → Dodger stadium → HK Macau → 🏠 San Gabriel garden inn

Note

中午時段前往Palm Springs Aerial Tramway，會先經過收票亭收費USD15的停車費，進停車場後再到售票處購買纜車票(15分鐘一班)，票務員會給你最近一班還有位置的纜車時間，接著就到旁邊的紀念品店、介紹區逛逛等時間到。纜車是旋轉纜車，所以站什麼位置其實不影響，搭乘時間大約10分鐘左右，到山上後就有許多步道可以健行，山上的氣溫也涼爽許多！下山就是直接排隊，沒有時間限制。

下山後我們前往Palm Springs市區開車逛逛看風景，然後打卡瑪麗蓮夢露的雕像，就前往LA了！我們先到飯店放所有行李後，再開車前往道奇球場，雖然停車費要USD35，但因為這次剛好遇到LA有衝突的時期，所以還是花錢買安心，就不搭大眾運輸了。前往道奇球場的只是非常清楚，入停車場處也都有人員驗票指揮停車位。唯一缺點就是，球賽結束時，要塞車一陣子才能離開球場。

因為我們住的區域華人比較多，所以附近也比較多亞洲餐廳，我們就挑了一間港式餐廳當宵夜。

Day9｜Ontario Mills - 安大略機場還車 - 安大略機場#

🍽️ Lucille’s smokehouse → Ontario Mills → 安大略機場還車 → 安大略機場

Note

退房後，前往Victoria Gateway Center Mall的Lucille’s smokehouse用餐，大概排隊半小時左右入座，有餐前麵包與抹醬，我們點Combo選Baby Back Ribs、Beef Brisket，整體還不錯拉，只是我個人是比較喜歡Texas Roadhouse。

然後開車去Ontario Mills，這是全室內的Outlet，這邊就比較沒有精品，是比較稍微平價一點的品牌集合，不過裡面有超多賣球帽、球衣的店！

安大略機場的還車處，旁邊就有接駁巴士的候車亭，巴士來的很快也大約10分鐘左右就可以到航廈！

Day10｜安大略機場 - 桃園機場#

總結#

聖地牙哥建議可以跟洛杉磯放一起玩，大概待3天左右就差不多！Palm Springs+Joshua Tree大概也是2-3天差不多，這兩個地方很適合一群人租Airbnb烤肉玩水！很推薦在美國看棒球，不論是哪個球場，整體的氛圍感受都很棒！
