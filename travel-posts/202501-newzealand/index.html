🇳🇿 紐西蘭

3 min

Time：2025/01#

✈️ 航班與入境#

去程：TPE → AKL → CHC
回程：CHC → ROT，AKL → TPE
紐西蘭電子旅行授權（NZeTA)
紐西蘭國際旅客保育及旅遊稅
紐西蘭入境卡

🗓️ 行程#

Day1｜桃園機場 - 奧克蘭機場#

桃園機場 → 奧克蘭機場

Note

第一次搭紐航，總共有兩餐、每人有兩件行李額度，直飛到奧克蘭機場大概11小時左右。

Day2｜奧克蘭機場 - 基督城機場 - 基督城市區#

奧克蘭機場 → 基督城機場 → 基督城市區晃晃：mona vale garden park、christchurch arts centre、Armstrong Lawn、bridge of remembrance、Cardboard Cathedral → 🏠 sudima hotel christchurch

Note

直飛到奧克蘭之後，要轉機到基督城，需要先提領行李後，再到國內航廈托運行李(走路大概10分鐘左右)，幾乎都有自助托運機器，也都會有人員在旁協助。

抵達基督城機場後，是安排基督城市區遊。先到mona vale garden park看看花與雅芳河，裡面有一個咖啡廳可以喝咖啡小憩一下。再到藝術中心跟植物園逛逛，接著去戰爭紀念橋跟紙教堂逛逛。

Day3｜tranzAlpine Train - Hokitika beach - Westland Tai Poutini National Park - lake matheson - Glacier Valley Cycleway Walkway#

Castle Hill → Arthurs pass station → 景觀火車tranzAlpine Train 1.5h → Greymouth station → Hokitika beach → Westland Tai Poutini National Park → lake matheson → Glacier Valley Cycleway Walkway → 🏠 Heartland Hotel

Note

Castle Hill有一些步道可以健行，也可以當作中途停靠點下車小逛一下。接著我們是從中段Arthurs pass station開始搭火車，因為現在是夏天，所以沿途看的風景大概就是山、湖、草、牛、羊等等這類的風景～然後一般是坐在座位區車廂，也可以走到站立區車廂隔著護欄看窗外的景色。

Westland Tai Poutini National Park跟lake matheson都是一小段步道，有需要經過該路段的人，可以當作休憩點，下去走走散步。

Glacier Valley Cycleway Walkway是在飯店附近的一條圓形小徑，裡面有超級多藍光蟲，不過裡面真的很黑，建議是一大群人一起去比較安全。

Day4｜冰川直升機 - Lake Hawea - Wanaka Tree#

冰川直升機 → Lake Hawea → Wanaka Tree → 🏠 oakridge resort wanaka

Note

一早來搭冰川直升機，我們是搭Fox Glacier Guiding - Glacier Tours & Adventures這家的行程，一人575NZD(可以現金也可以刷卡)。付款、秤重量、稍做解說後，會有接駁車載我們到附近的停機坪等候，並且聽從現場人員安排座位順序上直升機。起飛後會先飛到franz josef glacier冰川，接著降落在fox glacier 冰川上大約5分鐘，讓乘客們自由拍照，飛行員也會下來協助拍照，接著就再搭乘直升機回程，全部大約40分鐘左右。

Lake Hawea可以作為去瓦納卡小鎮的中途休息點，有一段大約3分鐘的步道可以爬上去欣賞湖的全景。

瓦納卡小鎮當然要去最有名的Wanaka Tree踩點，不過如果想要拍到漂亮的樹，要研究一下天氣在滿潮的時候來，樹才會孤獨的在湖中，不然旁邊的石頭泥巴也都會一起露出來。整個小鎮的感覺蠻悠閒的，也有一些紀念品可以購物，價位沒有特別貴。

Day5｜皇后鎮 - 箭鎮#

瓦納卡⇀皇后鎮搭纜車Gondola⇀🍽️ 景觀餐廳Stratosfare buffet⇀Skyline Luge⇀Arrowtown箭鎮⇀皇后鎮市區逛逛 → Queenstown Beach → 🍽️ Bakery → 🍽️ Fergburger → Cookie Time → 🏠 Millennium Hotel Queenstown

Note

終於來到大名鼎鼎的皇后鎮!!!!當然要來體驗南半球最陡的纜車(實際搭乘沒啥感覺)，不過纜車上看下去的風景真的很漂亮！抵達山上的纜車站後，步行大約2分鐘就可以到景觀餐廳Stratosfare buffet用餐，品項算多，餐點也蠻好吃的。好像因為在整修的關係，大約從餐廳門口步行8分鐘可以抵達Luge溜溜車的售票處與搭乘車。要從這邊再搭乘像滑雪的那種開放式纜車到達更上方的起點！整體車道不算長，但是蠻陡的，很好玩！個人覺得玩三次就差不多了，除非你真的很熱愛才需要買到五次。 (*可以在一樓買Gondola的時候就把餐廳跟Luge的票都一起買好，還在猶豫的話，也可以吃飽再單買Luge的票)。

接著前往掏金小鎮箭鎮，可以從停車處走到以前人掏金的河流，水很冰！也會看到遊客攜帶小工具進行掏金活動，不知道掏不掏得到就是了哈哈哈哈。整個小鎮不大，大約在這邊停留1-2小時就很足夠。

傍晚再次回到皇后鎮，我們到鬧區的Queenstown Beach看夕陽，蠻漂亮的。接著走到Fergburger，我們大約是21:30左右去買，排隊約10分鐘，等餐也大概10分鐘，漢堡皮是一般早餐店會有的漢堡皮，但是肉煎的很香，搭配生菜跟配料，整理蠻搭的！也可以趁等餐時間到隔壁的Bakery去買有名的甜甜圈，他是內餡包卡士達奶油的，整體蠻好吃的。

最後到鎮上Cookie Time的專賣店買了有紐西蘭全地圖的鐵盒餅乾，看過其他紀念品店跟機場，這家是最便宜的NZD25，因此如果有打算購買的人，可以在這邊就先購入了。

Day6｜TSS恩斯羅號蒸汽船 - Jet Boat#

Patagonia chocolaterie → TSS恩斯羅號蒸汽船 → walter peak farm → 🍽️ 午餐BBQ → Jet Boat → 🍽️ Devil Burger → 🏠 Millennium Hotel Queenstown

Note

早上因為有點陰雨綿綿，所以我們先到Patagonia chocolaterie喝咖啡吃巧克力冰淇淋等放晴，大約一小時雨也停了，剛好也差不多時間要去搭TSS恩斯羅號蒸汽船，是在TSS Earnslaw - RealNZ這個地標處集合上船。大約40分鐘後，我們靠岸在Walter Peak Station Wharf，這邊是一個小農場可以用餐也有綿羊秀和些許紀念品店。綿羊剛被剃下來的毛摸起來竟然油油的耶！

接著下午剛好比較晴朗，我們就參與了Jet Boat，會開一台噴射小船載大家，沿途風很大又很冷，其實一直在吹風，建議要戴毛帽跟防水外套跟穿雨衣!!!

Devil Burger的薯條一份很大盒，漢堡整體也蠻搭的，算是好吃，但沒有到超級驚艷這樣～

Day7｜Eglinton valley viewpoint - Milford Sound Marina - lake ta anau - Glow Worm Caves#

Eglinton valley viewpoint → mirror lake → monkey creek → Milford Sound Marina → lake ta anau → Glow Worm Caves → 🏠 the village inn

Note

Eglinton valley viewpoint是一個沿途的下車休息點，一大片的風景很適合下來散步。mirror lake在沒有風跟鴨子干擾的情況下，會有很完整的山倒映在湖面上。monkey creek也是一個簡單的小步道可以爬上去看風景。

接著在Milford Sound Marina登船，船上有簡易的咖啡吧！整趟航程大約1個半小時左右，一樓是座位區，下層是廁所跟座位區，上層可以到甲板上面吹吹風看風景。如果遇到前兩天有下雨的話，才有機會看到比較漂亮的瀑布，像我們這次是大晴天所以只有看到很小條的瀑布，但是整體峽灣的地形還是很讓人驚艷。

蒂阿腦藍光蟲則是在RealNZ - Te Anau Visitor Centre集合上船，會先有一艘船大約航行30分鐘左右到Glow Worm Caves，靠岸後會先帶大家到一個解說室，向大家介紹藍光蟲的相關資料。接著走進洞穴裡面，導覽員會指引大家走到船隻上，接著就是全黑的洞穴看藍光蟲了！！！

Day8｜Lake Pukaki - 庫克山隱士廬飯店#

The Stoaker Room Cromwell → ake Pukaki → Peter’s Lookout → 🏠 the hermitage hotel

Note

今天午餐是到The Stoaker Room Cromwell用餐，座位區是半開方式的戶外燒烤餐廳，食材是用木製酒桶煙燻所製作。最一開始會先上一份麵包佐特製調料醬與橄欖油讓大家分食，接著就是主餐的部分有雞、豬、牛、鮭魚、馬鈴薯、花椰菜、芝麻葉，整體都蠻好吃的！！

接著到Pukaki湖，這邊有一間商店Mt Cook Alpine Salmon Shop是專門賣鮭魚的店，鮭魚很新鮮，超級好吃，推薦喜歡大家有到這邊來都可以嚐鮮！！！再往前開一點會到Peter’s Lookout，這邊也很漂亮可以下來走走拍照及上廁所。

接著就是到此趟最期待的隱士廬飯店拉!!!!飯店位置離冰川其實還有一段距離，如果是住在高樓層比較有機會看到完整的庫克山景，當然也要當天天氣晴朗拉！隱士廬的自助餐蠻好吃的，選擇很豐富多樣化，甜點也很好吃。

Day9｜塔斯曼冰河船 - 蒂卡波湖&牧羊人教堂#

搭乘塔斯曼冰河船 → 蒂卡波湖&牧羊人教堂 → 🏠 sudima hotel christchurch

Note

一早先在飯店門口集合，會有接駁車接送至步道口，大約走路20分鐘就可以到搭船處，會先拍照穿救生衣很基本的解說後，讓大家依序上船隻，然後每艘船會有一位英文解說員，行駛中他會邊介紹冰川的時間、顏色、何時成形等等相關資訊，也會停在幾個比較特殊的地方幫大家拍照。整個行程我覺得超級推！建議大家可以在確定要去紐西蘭旅行之後，就先預訂這個行程!!!

Day10｜基督城機場 - 羅吐魯阿機場 - 紅木森林 - Rotorua纜車#

基督城機場 → 羅吐魯阿機場 → 紅木森林Redswoods → Rotorua Gondola → 🍽️ 景觀餐廳Stratosfare Restaurant & Bar → Skyline Luge → 🏠 Novotel Hotel

Note

今天抵達Rotorua之後，就先前往紅木森林Redswoods走走，接著到Rotorua的skyline搭纜車跟溜溜車，這邊的纜車自助餐廳也很不錯吃，不過我覺得最推的是這邊的溜溜車!!!佈景用得很好，車道也很長，推薦可以玩個三次！

Rotorua的Novotel飯店有大眾溫泉池、spa池跟小游泳池(需要穿泳衣的)，如果有機會住的話，非常推薦下樓泡泡湯！

Day11｜毛利文化村 - Lake Rotorua - Government Gardens#

毛利文化村 → Lake Rotorua → Government Gardens → 羊駝被工廠 → 🏠 Novotel Hotel

Note

今天一早起床後到毛利文化村來參觀，除了園區裡面有養育奇異鳥，這次很幸運兩隻都是醒著，然後我們也都有看到奇異鳥本尊！接著是園區裡的間歇噴泉跟地形也很特別！還有一小段毛利人的舞蹈表演。我們午餐也是在這個園區裡面用餐，這家餐廳是這幾天吃到最符合台灣人口味的餐廳XD。接著是到Rotorua湖去餵海鷗還有到Government Gardens去走一走看溫泉的源頭。

Day12｜Auckland Aquarium - mount eden - 奧克蘭sky tower#

Auckland Aquarium → okahu bay → mount eden → 奧克蘭sky tower → fortuna buffet restaurant → 🏠 Sudima Auckland Airport

Note

奧克蘭水族館裡面比較有特色的動物是企鵝、鯊魚、海龜，其他就跟一般的水族館差不多。 水族館出來門口就是okahu bay，可以稍微散步一下，但沒有到很特別。

接著是到mount eden是死火山口，停好車後大概要爬15分鐘的陡坡上去到火山口，環繞著火山口還有一大圈步道可以邊散步邊看奧克蘭市區的風景。

sky tower也是上去看風景的，看完風景後就到fortuna buffet restaurant用餐。

Day13｜奧克蘭機場 - 桃園機場#

奧克蘭機場 → 桃園機場

總結#

推薦的伴手禮：綿羊油相關產品(送人自用都不錯)、UMF™ 25+的蜂蜜、羊駝被、羊駝圍巾、cookie time餅乾有單獨包裝送人很適合。

推薦的景點：冰川直升機、塔斯曼冰河船、皇后鎮、米爾福德峽灣船、瓦納卡小鎮、第阿腦螢火蟲洞、Rotorua Skyline Luge、mount eden。

紐西蘭幾乎所有地方都可以刷卡，非常方便，如果要攜帶現金備用的話不需要準備太多。
