🇯🇵 沖繩

1 min

Time：2024/11#

✈️ 航班與入境#

去程：TPE → OKA
回程：OKA → TPE
此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code

🗓️ 行程#

Day1｜桃園機場 - 沖繩那霸機場#

桃園機場 → 沖繩那霸機場 → 🍽️ 豬肉蛋飯糰 → 飯店check in → 🍽️ 晚餐壽司郎 → 🏠 SMILE HOTEL

Note

下飛機就趕快衝，很幸運剛好前面都沒有人，所以第一組通關。豬肉蛋飯糰據說機場店好像比較不用排隊，所以入境後趕快往國內線的方向走，也蠻幸運15分鐘就買到！因為還先去飯店報到，所以比較晚才到國際通覓食，大部分開著的是居酒屋跟燒肉，但是怕不好消化，只好去吃壽司郎(個人是覺得跟台灣吃起來差不多)。

Day2｜沖繩中南部一日遊 - 國際通#

🍽️ 早餐全家 → klook一日遊：知念岬公園、沖繩世界、系滿海鮮市場、Ashibinaa outlet、賴長島 → 國際通逛逛買買 → 🍽️ 七輪燒肉 → 🏠 SMILE HOTEL

Note

今天是參加klook中南部一日遊，集合地點在RYUBO百貨的BLUE SEAL店前，遊覽車窗戶前面會放EASY GO的牌子，導遊也會在車門等，蠻好認的!! 沖繩世界的玉泉洞園區蠻直得去逛逛的！系滿海鮮市場是採自助式，自行去購買以及找座位、收餐盤等等。Ashibinaa outlet裡面有很多店家可以逛，有接駁車可到iias 沖繩豐崎(因為剛好遇到日本感謝祭，所以搭過去去uniqlo爆滿一波)。瀨長島蠻chill，蠻推薦到那邊喝下午茶看海與飛機起降。

國際通整條走走看看的話，大概要25分鐘，街上大部分是伴手禮店與藥妝店，晚上蠻熱鬧的，可以去走走逛逛！七輪燒肉是比較靠近牧志站那邊，是屬於平價燒肉，每份食材也都是單點的形式，當天大概是7點左有抵達，稍微排隊了15分鐘才進去用餐，這家CP值蠻高。

Day3｜觀光玻璃船 - 美國村#

🍽️ 早餐全家 → 觀光玻璃船 → 🍽️ 午餐蕎麥麵 → 搭公車去美國村 → 🍽️ 琉家拉麵 → 🏠 SMILE HOTEL

Note

觀光玻璃船如果你是不敢浮潛然後對像浮潛看魚那樣有興趣的話，可以去體驗看看，當天在櫃檯報到之後，會由專員帶領上船，接著等船稍微開遠一點之後，工作人員會廣播，請大家到下船艙去，下去之後，可以坐在椅子上從船側邊，看海底的珊瑚跟魚群，當天能見度是寫level 2，不過還是看到蠻多魚群的！

午餐是到那霸巴士總站附近的沖繩料理吃蕎麥麵，很好吃！接著就從那霸巴士總站搭乘巴士到美國村，大約1小時的路程，在美國村逛逛、吃BLUE SEAL 冰淇淋，悠閒的度過下午。

晚餐則是回到國際通街上吃琉家拉麵，我們大概6:30左右到，還沒有人排隊，馬上進去吃，很好吃，而且也可刷卡！

Day4｜波上宮 - 珊瑚畫染 - 沖繩那霸機場 - 桃園機場#

🍽️ 早餐全家 → 波上宮 → 珊瑚畫染 → 🍽️ 小祿站通堂拉麵 → 那霸機場 → 桃園機場

Note

波上宮蠻小的，大約20分鐘可以參觀完畢，旁邊有步道可以走去海邊，水很清澈、沙灘也很白，若是夏天還可以下去玩玩水！

接著是去珊瑚畫染，體驗方式是將顏料沾在包著珊瑚模具的包包或是衣服上面，讓顏料繪製出圖案上面，是一個蠻適合長輩與小孩的體驗活動。

回程到單軌小祿站吃通堂拉麵，這站也有商場跟大國藥妝可以逛，是大家回程搭機前的好選擇！

總結#

沖繩真的蠻適合帶長輩或小朋友去玩！行程上不會太緊湊，租車也算蠻方便加便宜，若是不自駕住在那霸市區參加一日遊or搭大眾運輸也還算方便(只是有時候公車時刻表不太準)。這次11月底去，天氣很舒服，沒太陽時就會涼涼的，也不會太冷，穿短袖配外套。
