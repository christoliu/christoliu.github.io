🇲🇾 蘭卡威

2 min

Time：2024/06#

✈️ 航班與入境#

去程：TPE → SIN → LGK
回程：LGK → KUL → TPE
此次旅程時新加坡與馬來西亞皆免簽 72小時內上網填寫【新加坡電子入境卡SG Arrival Card】

🗓️ 行程#

Day1｜桃園機場 - 樟宜機場 - 蘭卡威機場 - 訂跳島行程 - Hidden lankawi - 按摩 - 夜市#

桃園機場 → 樟宜機場 → 蘭卡威機場 → 飯店 → 訂行程 → 買伴手禮 → 🍽️ 晚餐Hidden lankawi → 按摩 → 🍽️ 夜市 → 🏠 CENANG PLAZA BEACH HOTEL

Note

搭紅眼班機抵達新加坡後有大約5個半小時等待轉機，我們決定入境新加坡去逛星耀樟宜。因為T1轉機區的亞坤在整修，所以我們是入境後先到T3吃亞坤，接著十點半左右去看瀑布&逛逛百貨公司，最後去B2吃松發肉骨茶，就出境回到候機室了(出境時只有快速通關刷護照、檢查行李跟身份是在登機口)。

蘭卡威機場蠻小的，抵達後排隊查驗護照大約20分鐘左右，接著我們在機場門口與提早於易遊網預定的機車行碰面去牽機車(易遊網只告訴我們會有人舉牌接我們，我們到門口後完全無法與對方聯繫上，後來是其他商家協助聯繫的)。機場大廳有很多商家會問你要不要租車，價格也比較便宜，大家也可以考慮到當地後直接租車。

放完行李後走到附近的AB Tour在AB Motel門口，我們預定了跳島行程(孕婦島、巨獅島、溼米島)。因為距離日落還有一點時間，所以我們先到飯店附近的伴手禮店買了一些歐米呀給，蘭卡威整座島都是免稅的，所以買伴手禮蠻便宜的。

騎去Hidden lankawi吃晚餐看日落，這家餐廳在沙灘旁邊，可以直接走下去沙灘、也有駐唱歌手、座位有戶外跟室內，雖然都沒有冷氣，但是傍晚去的時候不會很熱～～

吃飽後到ALUN-ALUN spa，這是我們先在KLOOK上預定的，不過預定時沒讓選店家，但蘭卡威有四五間分店，所以我們還是到現場詢問是否有時段。選擇1小時的精油按摩，我跟璇按起來的感覺是按摩師只是順著肌肉推油，沒有特別有感，而且按摩師還一直講悄悄話嬉鬧。

回飯店後順著珍南大街晃晃，遇到一個小夜市買了點宵夜回飯店吃，結束第一天。

Day2｜跳島行程 - 落遊艇 - 瓜鎮夜市#

🍽️ 早餐星巴克 → 跳島(9-13) → 🍽️ 午餐Fat-Cupid → 騎車去瓜鎮碼頭集合 → 日落遊艇(16-20) → 🍽️ 瓜鎮夜市 → 🏠 CENANG PLAZA BEACH HOTEL

Note

起床後到對面cenang Mall一樓的星巴克買早餐，這邊售價大約是台灣的7折。然後到A B Tour集合出發去跳島行程，店家會開一台小箱型車載我們到搭船的地方，搭快艇到第一個孕婦島停留1小時，爬一段紅樹林步道大約10分鐘，會有一個湖可以游泳跟租借腳踏船、獨木舟、SUP等等(想玩的人記得帶現金)>第二個島巨獅島，這邊有一大群老鷹盤旋，開船者會餵食老鷹>第三個島濕米島，這邊有白沙灘也可以租借基本的水上設施香蕉船、滑翔傘和摩托艇等等(記得帶現金)。

解散之後，我們騎到Fat-Cupid吃招牌的叻沙麵，喜辣者應該會很喜歡～不過這家餐廳有遮陽跟風扇但一樣沒有冷氣，所以吃完有點冒汗～稍作整頓就騎車去瓜鎮碼頭集合。

四點準時登船，船上有無限供應的飲料調酒，大約開船15分鐘後商家會放像漁網的設施拖在船尾讓大家可以穿救生衣下去泡海水，15分鐘後會收漁網換成開始玩水上設施SUP、溜滑梯、跳船等等，玩水1小時後上船吃晚餐等看日落，這時候商家會帶動氣氛，整艘船會很嗨～～

下船解散之後，我們騎車到附近的瓜鎮夜市買晚餐回飯店吃，結束第二天。

Day3｜騎車環島 - 瓜鎮 - 按摩 - 蘭卡威機場 - 吉隆坡機場 - 桃園機場#

🍽️ 早餐Red Tomato → 騎車環島 → 🍽️ 瓜鎮Chicken Joe海南雞飯/Restaurant Shark Fing肉骨茶 → 瓜鎮買伴手禮 → 按摩 → 機場還車 → 搭飛機 → 吉隆坡轉機 → 桃園機場

Note

早餐去吃Red Tomato後開始一日環島，9:40出發環島 -30m- 東方村&纜車 -25m- 黑沙灘Pantai Pasir Hitam -10m- PANTAI TANJUNGRHU -20m- KIRIM GEOFOREST PARK -25m- 瓜鎮Chicken Joe海南雞飯/Restaurant Shark Fing肉骨茶/parade伴手禮 -30m- 16:45抵達Yuan spa，這家很熱門要提前去預約時間，實際去過之後真的很推薦!!!

我們大約提前1.5小時到機場，先吃點晚餐，等登機時間前20分鐘才進去安檢，機場很小，出口門馬上就接著安檢，然後走3分鐘就是登機門了。

我們在蘭卡威報到就把兩段機票都拿好了，所以抵達吉隆坡T1時，就沿著轉機指標，會到一個轉機櫃檯，人員幫你蓋出境章後，下樓去搭接駁巴士，大概搭5分鐘後，就到我們的Gate大樓，走上去就是候機室了，檢查行李跟身份一樣是在候機室門口，最後登機結束此次快閃旅程拉～～～!!

總結#

雖然要轉機，不過蠻適合請一天假快閃的小資族，島上消費不高

天氣的部分，28-32度偏悶熱、太陽很大記得塗防曬、沒太陽的地方風是涼的，大多是穿背心/長褲/防曬外套/帽子/太陽眼鏡

店家幾乎都可以刷卡，大概只有夜市跟攤販會用到現金，也要注意租車跟飯店押金有些也只能現金，我們這次實際需要現金大概150馬幣加上押金150馬幣
