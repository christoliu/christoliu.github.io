🇯🇵 東京-夏

1 min

Time：2023/07#

✈️ 航班與入境#

去程：TPE → NRT
回程：NRT → TPE
此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code

🗓️ 行程#

Day1｜桃園機場 - 成田機場 - 雷門淺草寺 - 晴空塔 - 上野居酒屋#

桃園機場 → 成田機場 → 日暮里check in → 雷門淺草寺 → 晴空塔 → 🍽️ 上野居酒屋 → 🏠 日暮里APA

Note

抵達領好行李後，因為我們的skyliner票是在klook上面訂，所以要先到skyliner櫃檯換實體票，櫃檯會給你最近一班車的車票及座位號，然後櫃檯不遠處就是乘車入口，進站後就對號上車即可。

到飯店check-in後，就前往雷門淺草寺逛逛，雖然是平日來訪，但還是人潮擁擠呢！幾個推薦小點心：木村家人形燒本舖、淺草炸肉餅(浅草メンチ)、炸牛排，可以買來當下午茶吃。另外也很多人會在這體驗和服，但因為我是夏天去太熱了，所以就沒體驗～

傍晚我們前往晴空塔，若是已經在網上訂購門票，則出示QR畫面給工作人員核對就好！蠻推薦傍晚時刻來，可以一次看到白天、夕陽、夜景的景色～

晚上我們到上野阿美橫丁與朋友聚餐，朋友挑了當地人吃的居酒屋，菜單都是日文，也只能用日文點餐，很酷的體驗。若是有日文稍好的朋友到阿美橫丁也可以挑這種當地人吃的居酒屋試試喔！

Day2｜shibuya sky - 銀座 - 增上寺#

涉谷交叉點 → shibuya sky → 🍽️ 三本漢堡排 → 銀座 → 增上寺 → 🍽️ 上野壽司三味 → 🏠 日暮里APA

Note

早上先到涉谷交叉點參觀一下人來人往的大十字路口，接著到shibuya sky看風景，建議可以在網路上訂票，買接近日落的時間，應該會更漂亮！

然後走在路上被排隊的漢堡排吸引進去，果然是看到排隊就想排一下哈哈哈哈哈哈，大概排了40分鐘，但真的蠻好吃的！然後就到銀座去逛街了，銀座週末會把車道封起來變成人行道，讓行人可以更安全的逛街～這邊店家很多，也有很大間的Uniqlo可以逛！

增上寺是拍東京鐵塔很不錯的位置，大家可以到這邊散步加拍照！然後我們就在上野找的一間看起來不錯的壽司店當晚餐拉～在日本好像吃壽司不太會踩雷！

Day3｜迪士尼#

迪士尼海洋 → 🍽️ 馬賊 → 🍽️ 鳥貴族 → 🏠 日暮里APA

Note

一早在超商買早餐跟小點心後，就前往JR舞濱站，下車後若是要搭迪士尼的單軌車，記得要買一日票才會是漂亮圖案版本的車票！在門口排隊等待8:30開園，大約9:00左右進園區！

進園區後，我們的遊玩順序是：先用DPA買夢幻奇航 → 走去排隊驚魂古塔(大約排45分鐘) → 買DPA地心探險 → 10:20看第一個遊行眾彩同慶 → 玩夢幻奇航 → 11:30看第二個遊行達菲與好友 → 吃米奇包子、米奇雞塊 → 玩海底兩萬哩 → 地心探險 → 吃咖哩棒、三眼怪、雞腿 → 玩單人通道忿怒雙神 → 玩單人通道印第安納瓊斯() → 吃米奇冰棒 → 玩小胖的飛魚雲霄飛車 → 玩尼莫與好友的海洋世界 → 搭海洋電氣化鐵路 → 逛紀念品店買伴手禮 → 吃米奇吉拿棒、吃爆米花 → 看神燈劇場 → 玩辛巴達傳奇之旅 → 搭海洋渡輪航線 → 吃熱狗 → 20:00最後的遊行堅信夢想之海(海洋的遊行都是人物搭船在地中海港灣表演，所以晚上最後遊行時也有稍微放小型的煙火)。

回到住宿區附近先吃馬賊麵墊墊胃，再到鳥貴族吃去串燒、喝酒，解放一下玩一整天迪士尼的辛苦哈哈哈哈哈

Day4｜成田機場 - 桃園機場#

skyliner → 成田機場

Note

退房後，在附近買全家的豬排三明治(真的很好吃)，就搭skyliner從日暮里前往成田機場了！

總結#

七月中的東京比想像中悶熱，可以完全穿夏衣。這次旅程都是去東京比較有名的景點，首次前往東京的旅人們可以參考！日本迪士尼有兩個園區陸地跟海洋，第一次去迪士尼的話推薦去陸地，因為有城堡。但是達菲家族的商品只有海洋賣而且設施也稍微刺激一些，所以就大家各挑自己所喜囉～
