🇬🇷 希臘

3 min

Time：2023/04#

✈️ 航班與入境#

去程：TPE → SIN → ATH，ATH → JTR
回程：JTR → ATH，ATH → SIN → TPE
此次旅程歐洲申根國、新加坡皆免簽
新加坡入境要填新加坡電子入境卡 SG Arrival Card

🗓️ 行程#

Day1｜桃園機場 - 樟宜機場轉機 - 新加坡市區#

桃園機場 → 樟宜機場 → Tekka Center → 小印度區 → 小阿拉伯區 →🍽️ 白蘭閣街大蝦麵 → Suntec Citymall → jollibee → 吃茶三千 → 魚尾獅公園 → 🍽️ 松發肉骨茶 → 克拉碼頭 → 牛車水(Chinatown) → 天天海南雞 → 老巴剎 → 金沙酒店水舞 → 濱海灣植物園燈光秀 →🍽️ shake shack → sats lounge

Note

搭乘一大早抵達新加坡的飛機，將行李寄放在機場的寄物櫃檯後開啟新加坡一日遊，先到Tekka Center吃午餐，這邊有很多攤販跟公共座位區，自己買完餐點後找座位。接著到小印度/阿拉伯區，這邊有一些小店、特殊建築、牆畫可以走走看看，阿拉伯區也有地毯專賣店，很酷～接著到白蘭閣街大蝦麵點了一碗麵分食。

然後就到Suntec Citymall逛街吹冷氣拉！新加坡的白天真的超級無敵熱，請大家做好又熱又曬的心理準備XD 裡面有Jollibee快樂蜂跟吃茶三千，啊吃茶三千跟台灣價格賣一樣，但是新加坡物價比較高，所以在這邊喝算平價手搖飲～～～

接著我們去魚尾獅公園跟大名鼎鼎的魚尾獅拍照拉，然後去吃松發肉骨茶，店家裡面有冷氣，出餐也很快，餐點也真的好吃!!! 又到克拉碼頭、牛車水(Chinatown)去晃晃。天天海南雞是在市場裡面，可以分工一人去排隊買餐，一人去找座位，真的好吃！老巴剎則是有需要串燒店家跟露天座位，人很多，出餐都要等蠻久的。

金沙酒店水舞秀、濱海灣植物園燈光秀請排入行程裡面，作為晚上的休息去處，還蠻推薦的。回機場前吃了shake shack漢堡當宵夜～～就到貴賓室去休息拉！

Day2｜雅典市區#

樟宜機場 → 雅典機場 → 雅典憲法廣場 → kosta souvlaki → davinci gelato → 雅典學院 → 希臘國家圖書館 → 利卡維多斯山丘夜景 → 🍽️ tzitzikas kai mermigas → 無名英雄雕像記念碑 → 雅典機場

Note

從新加坡搭飛機大概11小時後抵達希臘雅典啦!!!!，前一天請安排多一點活動盡量消耗體力，這樣在飛機上才能多睡一點～因為隔天還要飛，所以抵達雅典後我們就將行李寄放在機場，然後搭X95公車前往市中心憲法廣場Syntagma Square，第一站就先去買souvlaki希臘式捲餅的概念，裡面會包肉、番茄、薯條，除了嚐鮮，若是旅費比較拮据，也可以多吃幾餐這個XD 飯後甜點去吃義式冰淇淋店 Da Vinci Gelato。

雅典學院我們只有在外面拍拍照，門口的雕像左側是柏拉圖Plato，右側是蘇格拉底Socrates，正門上方的浮雕描繪了希臘神話中雅典娜的誕生。隔壁的希臘國家圖書館 氣勢磅礴，螺旋大理石階梯是許多旅人拍照打卡的熱點。

傍晚前搭乘纜車登上 利卡維多斯山丘Mount Lycabettus，這裡是俯瞰雅典全城與遠方愛琴海的絕佳地點。搭配夕陽，真的很漂亮！

回到市中心，在人氣餐廳 Tzitzikas kai Mermigas 用餐，點了烘蛋跟海鮮燉飯，餐點好吃又不貴，CP值很高！晚餐後走到 無名英雄雕像紀念碑看衛兵交接儀式！因為是清晨的飛機，所以我們回到雅典機場休息等待。

Day3｜Fira - Oia - Fira#

雅典機場 → 聖托里尼機場 → 租機車moto panos → aroma suites check in → 騎車去oia →🍽️ skiza cafe → 回Fira →🍽️ lucky souvlaki → 飯店看夕陽 → 🏠 Aroma Suites

Note

一早搭乘1小時飛機抵達聖托里尼拉!!!下飛機後Moto Panos機車店的老闆就來接送我們，我們這幾天選擇租摩托車，辦理好手續後，就迎著微風上路，正式展開聖托里尼之旅！

第一站先到Fira入住期待已久的Aroma Suites。白牆藍窗、陽台望海，整體充滿典雅南愛琴海風格，整個好喜歡～～～稍微休息過後，我們騎車穿沿著懸崖騎向北方的夢幻小鎮Oia逛逛拍照！真的跟介紹一樣，藍白相間的房屋與無敵海景！在Oia找一間海景咖啡廳Skiza Cafe，邊享用美食邊欣賞懸崖下的愛琴海。

接著回到Fira買lucky souvlaki回飯店的陽台邊吃邊欣賞夕陽！

Day4｜Fira - Oia#

🍽️ 飯店早餐 → 逛Fira →🍽️ meat corner grill → art&icon studio → 騎車去oia → caldera villas check in → 🍽️ lotza → 家樂福買紅酒 → 🍽️ 甜點店melenio → 🏠 Caldera villas

Note

住在 Aroma Suites的好處，就是早上睜開眼就能看見藍天與無邊海景。飯店的豐盛早餐配著無敵風景！早餐後在Fira的小巷散步，這邊感覺算是聖托里尼熱鬧的一區。中午來到Meat Corner Grill吃午餐，吃飽後順路走進一間溫暖的小畫廊Art&Icon Studio，裡面有許多的藝術品，可以來尋寶！

午後再次騎摩托車，往北行駛，沿在海邊的風景前往Oia，抵達夢幻的Oia後，先到懸崖上的Caldera Villas Check-in，從房間陽台望出去，是著名藍頂教堂與火山灣！

傍晚時走進Oia巷子裡的Lotza Restaurant，位置極佳，可以邊吃邊看夕陽，我們點了烤魚跟義大利麵，味道都還不錯也不貴！飯後前往附近的小型 Carrefour，買了店員推薦的本地紅酒回飯店；再走去著名的甜點店Melenio Cafe，買開心果千層蛋糕！

太陽下山後，我們前往Oia Sunset Photo Spot欣賞整個Oia的夜景，山壁上的房屋配上燈火，非常好看！

Day5｜Oia - Kamari - Imerovigli - 聖托里尼機場 - 雅典機場 - 雅典市區#

🍽️ skiza cafe → 逛Oia → Oia post office → kamari beach → 🍽️ Happy grill → santo wines酒廠 → Imerovigli逛逛 →🍽️ Lucky souvlaki → 聖托里尼機場 → 雅典機場 → 🏠 Tempi Hotel

Note

早上到skiza cafe買了可頌與希臘優格淋蜂蜜回飯店享用！早餐後在 Oia 隨意閒晃，挑了幾張明信片，走去 郵局投遞。Oia的每條巷子都好漂亮，隨便拍都好看。接著我們就離開Oia來到東南岸的 Kamari Beach，這裡有整片的黑沙灘跟陽光，不過因為天氣還不夠熱，所以還沒開始海上活動！

在Kamari鎮上找了間可愛的餐廳Happy Grill吃海鮮燉飯與炸魷魚，很新鮮又好吃！離開Kamari後，來到當地的釀酒廠santo wines參觀。

在前往機場前，特地繞到寧靜得小鎮Imerovigli散步，這裡是是聖托里尼海拔最高的小鎮，位於險峭懸崖之上，面向著愛琴海及火山口Caldera。

回到Lucky’s Souvlaki，點了份豬肉Pita加炸薯條後就去還機車了，老闆一樣接送我們到聖托里尼機場。飛機抵達雅典，一樣搭X95公車前往市中心憲法廣場Syntagma Square，入住市中心的Tempi Hotel。

Day6｜雅典衛城 - 泛雅典運動場 - 哈德良拱門 - 聖托里尼機場 - 宙斯神殿#

雅典衛城Acropolis → 泛雅典運動場Panathenaic Stadium → 哈德良拱門Hadrian’s Arch → 宙斯神殿Temple of Olympian Zeus →🍽️ Feyrouz → 🏠 Tempi Hotel

Note

結束聖托里尼的海島度假，回到雅典，開始歷史的探索。一早就出發前往最有名的雅典衛城，這裡是西方文明的起點，也是最壯觀的古蹟。登上山丘，望見帕德嫩神廟Parthenon、瑞克提翁神殿Erechtheion、勝利神廟Temple of Athena Nike、普羅皮雷亞門Propylaea這些建築，好像親自到歷史課本上一樣。

走路約15分鐘，來到全世界唯一一座全大理石打造的泛雅典運動場。這裡曾是1896年第一屆現代奧運的主場，純白大理石在陽光下閃閃發亮，想像觀眾曾坐滿的景象，還蠻震撼的。

接著前往哈德良拱門，這座拱門恰好劃分了舊雅典與新雅典，它的存在代表著兩個時代的融合。拱門旁就是壯觀的奧林匹亞宙斯神殿。雖然只剩幾根巨柱聳立，但現場氣勢依然讓人屏息。天空很藍配上石柱，拍起來好好看。

中東風餐廳 Feyrouz，點了最有名的 lahmajoun，滿特別的食物，推薦大家可以去嚐嚐。

Day7｜Little Kook - Monastiraki Square - 古雅典阿哥拉 - 衛城博物館#

🍽️ Lukumades → Check in → Litte kook → Monastiraki square逛逛買草莓 → 古雅典阿哥拉Ancient Agora of Athens → o thanasis pita → 衛城博物館 → 🏠 Pella Inn

Note

一早來到Lukumades，這家超人氣的小店專賣希臘油炸甜甜圈，外酥內軟的甜點，撒上濃濃的蜂蜜與糖粉，很好吃！接著到Pella Inn check-in與放行李。接下來，走進了 Little Kook，這個咖啡廳有著超級浮誇的裝飾，有機會的話可以到裡面喝咖啡試試看！隨後來到Monastiraki Square逛逛，在雅典買到最高CP值的就是又好吃又便宜的草莓拉！

古雅典阿哥拉Ancient Agora of Athens這個曾經是古雅典的政治、商業與文化中心，這裡的 赫淮斯托斯神殿Temple of Hephaestus，也保存得相當完整。午餐來到O Thanasis吃希臘皮塔餅配汽水！之後我們到衛城博物館參觀。這座博物館收藏了大量與雅典衛城相關的藝術品、雕刻和歷史文物。

因為有點想念中式料理，所以就外帶了連鎖的炒麵店，回到旅館頂樓配夕陽享用！

Day8｜中央市場 - 雅典國家考古博物館#

🍽️ karaman lidika → cafe venetti → 中央市場 → 雅典國家考古博物館 →🍽️ efcharis → 🏠 Pella Inn

Note

午餐我們來到了Karaman Lidika，這是一家典型的家庭式希臘餐廳，我們點了烘蛋、希臘優格、涼拌魚，每一道都很有特色，這間餐廳也蠻推薦的！接著，我們來到Cafe Venetti我們點了拿鐵，還品嚐了當地的甜點(真的超甜)。隨後，我們稍微逛了一下中央市場，就前往雅典國家考古博物館，館內收藏了數以千計的古希臘文物！

晚餐我們到Efcharis用餐，這家餐廳提供傳統的希臘家庭菜餚，點了炸切達起司、海鮮義大利麵、葡萄葉捲飯，每一套都很特色又好吃，這間餐廳也很推薦！

Day9｜雅典市區 - 雅典機場 - 樟宜機場#

sintagma to airport(X95公車) → 雅典機場

Note

離開雅典市中心的時候，嘗試買了街上小攤販的蝴蝶餅當早餐，結果意外的超好吃欸！

sintagma to airport(X95公車)可以從市區直達雅典機場。

Day10｜樟宜機場 → 桃園機場#

早餐吃亞坤 → 逛機場內免稅店 → 桃園機場

Note

抵達新加坡在等待轉機之際，我們又去吃了一次亞坤，吃完之後就回到機場內採買一波CHARLES & KEITH，大概是台灣的8折左右。

總結#

新加坡機場是目前我覺得最棒的轉機機場，轉機安檢很方便快速，只有在登機門前安檢一次，就算轉機時間只有3-4小時，也可以入境新加坡到星耀樟怡去逛街、看瀑布和吃美食！

雅典市區真的要很喜歡歷史的人，不然可能待個2天就差不多了！聖托里尼真的很漂亮，不過比較適合夏天去，會有比較多水上活動可以玩，但房價也會變得超級貴XD
