🇯🇵 北海道

2 min

Time：2022/12#

✈️ 航班與入境#

去程：TPE → CTS
回程：CTS → TPE
此次旅程日本免簽需要先上網填寫Visit Japan Web，抵達時出示QR code

🗓️ 行程#

Day1｜桃園機場 - 新千歲機場 - TOMAMU#

桃園機場 → 新千歲機場 → 南千歲JR站 → TOMAMU車站 → 星野接駁車接送至飯店→ 🍽️ Soup King Curry → 微笑海灘 → 🍽️ 森之餐廳 → 愛絲冰城 → 水之教堂 → 林木之湯 → 🏠 TOMAMU

Note

抵達新千歲機場後，就直接前往巴士搭乘處搭巴士到南千歲等搭JR，巴士下車處要走一個空橋到南千歲JR站，大約搭乘1.5h後抵達TOMAMU站，會有星野的接駁巴士在這邊等大家。

飯店check-in後，就可以搭度假村裡面的接駁巴士前往各景點拉！我們先晃到美食區去吃Soup King Curry，蠻好吃的，價格也不算太貴！吃飽後去微笑海灘玩水，裡面很溫暖有人造海浪跟spa池，跟外面的冰天雪地很反差～

晚餐到森林餐廳去用餐，餐點我們個人覺得普通，但是餐廳的氛圍感很好，所以還是推薦去吃！愛絲冰城有很多冰塊做的房子跟溜冰場，但我們來的時候溜冰場還沒弄好，所以就先在冰塊屋吃東西烤棉花糖，然後晚上這邊也會放音樂跟煙火(要先查一下時刻表)。接著走到水之教堂去參觀。

林木之湯很推薦大家去泡，是男女分開的露天風呂，身體泡在45℃熱水，頭在-15℃欣賞森林雪景

Day2｜TOMAMU#

霧冰平台 → 滑雪 → 🍽️ Jingisukan KING → 愛絲冰城 → 林木之湯 → 🏠 TOMAMU

Note

飯店有發早餐券，但因為我們還有自己買的小食，所以今日早餐就吃自己準備的。早上搭纜車前往霧冰平台散步、拍照，大概可以待3小時！櫃檯可以詢問教練課的資訊，但因為我們只有一個下午所以就自己租裝備在比較低的區域滑雪了！

晚餐拿早餐券去吃Jingisukan KING烤肉(可以查一下星野裡面的餐廳哪些可以抵用)，這家也好吃，蠻意外度假村裡面的食物，其實都還不錯吃！接著又到愛絲冰城去逛逛，然後去林木之湯！

Day3｜TOMAMU - 札幌#

🍽️ Buffet Dining hal → 星野接駁至TOMAMU車站 → 時計台 → 札幌電視塔 → 北海道神宮 → 藻岩山夜景 → 🍽️ 信玄拉麵 → 🍽️ 鰻魚飯うなぎ二葉 → 札幌大通公園(札幌燈節) → 札幌慕尼黑聖誕節市集 → 唐吉軻德 → 🏠 Grids Sapporo

Note

早上選擇來吃Buffet Dining hal，這個餐廳的風景很好，早餐也蠻好吃，作為最後一天的早餐很不錯！接著到櫃檯登記接駁車至TOMAMU車站，就離開星野前往北海道市區拉～

到市區飯店check-in後，就前往時計台拍照，然後到札幌電視塔看風景，這邊也有一些紀念品小物可以逛逛，還有一台拍貼機可以玩。

北海道神宮，是一個蠻大的公園，除了神宮參拜之外，也可以散步，然後剛好有遇到其他民眾在樹幹和椅子上做雪雕，還蠻可愛的，大概可以花2小時在這邊逛逛、玩雪，也有神宮茶屋可以吃下午茶。

藻岩山夜景是北海道三大夜景之一，如果從路面電車直接走到藻岩山麓站，大概8分鐘，可以現場買票。纜車運行分成兩段，山麓駅→中腹駅→山頂駅(展望台)，我們是直接買到展望台！第一段大概5分鐘，比較漂亮，可以盡量到纜車車廂往山下的方向，縮時錄起來蠻好看的！第二段時間很短也看不太到風景，所以位置上就比較沒有講究。展望台可以看到整個札幌城市夜景，因為沒有高大的建築物或是其他自然景觀，所以就是一大片的燈火通明，我覺得拍起來很漂亮！

回到市區後，我們先到信玄拉麵去填飽肚子後，就去逛札幌大通公園得札幌燈節跟札幌慕尼黑聖誕節市集，剛好遇到下大雪，所以整個更有氛圍感！

宵夜就在飯店吃稍早外帶的鰻魚飯うなぎ二葉，日本的鰻魚飯真的是蠻不容易踩雷的！

Day4｜旭山動物園 - 富良野 - 白鬚瀑布 - 森林精靈露台 - 札幌市區#

札幌東急REI飯店集合 → 旭山動物園(120分鐘) → 富良野Herb Garden午餐(60分鐘) → 白鬚瀑布(20分鐘) → 森林精靈露台(30分鐘) → 札幌慕尼黑聖誕節集 → 貍小路商店街 → 🍽️ 串鳥居酒屋 → 🏠 Grids Sapporo

Note

今天要參加Klook的旭川一日遊，一早我們先到名代飯糰外帶當早餐，再去集合地點集合，遊覽車跟導遊都已經在集合點門口等，只要報上姓名，就可以到遊覽車上等出發拉！大概車程2小時，就會到第一站旭山動物園，在門口下車領門票後，就自由時間2小時，大家可以自由安排想逛的區。可以先問導遊企鵝遊行的時間，先到場區佔位置，企鵝會近距離的出來散步給你們看！接著我們就到北極熊、黑熊、長頸鹿、鳥類區去參觀，今天剛好遇到下大雪，所以整個動物園都白白一片，逛起來特別有趣。

午餐是安排到富良野Herb Garden，早完後旁邊也有紀念品店可以買薰衣草相關的商品！白鬚瀑布大概停留20分鐘左右的時間，除了中間的小溪是藍綠色的之外，兩旁都已堆滿了雪，連水流也都結冰，蠻酷的！

森林精靈露台是今日最後的景點，剛好是傍晚抵達這邊，天色漸黑加上露台的點燈，讓這邊看起來超級無敵漂亮，怎麼拍都好好看！

回程我們選在札幌電視塔下車，再次前往聖誕市集去吃東西跟喝熱紅酒，這個市集加上下大雪，真的很有歐洲的聖誕市集氛圍感！

貍小路商店街有很多藥妝店跟伴手禮可以購物，也有很多餐廳，我們選擇到串鳥居酒屋去吃串燒、喝啤酒作為今日的結尾。

Day5｜二条市場 - 買伴手禮 - 新千歲機場 - 桃園機場#

🍽️ 二条市場 → 買六花亭→ 🍽️ 一蘭拉麵 → 新千歲機場 → 桃園機場

Note

早上我們到二条市場去吃海鮮，因為沒有找到可以代客料理帝王蟹的店家，所以最後我們找了一間環境看起來還不錯的餐廳去吃海鮮丼飯，這邊因為專賣海鮮，所以海鮮料理的店家食材吃起來都蠻新鮮的！

我們到北海道有名的伴手禮店六花亭去採購一波，再去吃一蘭拉麵當午餐！就搭JR到新千歲機場拉！我們是抓2小時前到新千歲機場，時間上還算充裕。

總結#

就算不滑雪、帶長輩小孩出門，我也推薦來體驗星野TOMAMU度假村，整個度假村有很多設施可以玩，園區內的巴士路線也都很清晰，每個設施的體驗也都很不錯！札幌市區的話推薦住在貍小路商店街附近，離逛街、吃飯跟幾個主要景點都不遠！另外，可以查一下札幌慕尼黑聖誕節市集的時間，剛好有遇上的話也蠻推薦去逛一下，尤其如果遇到下雪就更有氛圍了！
