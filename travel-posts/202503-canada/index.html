🇺🇸 🇨🇦 西雅圖/加拿大

3 min

Time：2025/03#

✈️ 航班與入境#

去程：TPE → SEA，YVR → (YYC) → YZF
回程：YZF → YYC，YYC → SEA，SEA → TPE
此次旅程使用上次申辦的ESTA入境(期效3年)

🗓️ 行程#

Day1｜桃園機場 - 西雅圖機場 - T-mobile球賽#

桃園機場 → 西雅圖機場 → check in → T-mobile球賽 → 🍽️ Denny’s → 🏠 Red Roof Inn

Note

西雅圖機場是先領行李後再過海關入境，等行李大概15分鐘，然後這時間完全沒人排隊海關，所以我們從下飛機到離開機場應該不到半小時。因為住在機場附近的飯店，所以跟著Hotel Shuttle巴士的指標就可以到等車處，這邊有各飯店的接駁聯繫電話表以及一台電話，但是我們去的時候，電話上面貼暫停使用，所以我們還是用自己的電話撥打飯店，請接駁車接送了。

從西雅圖輕軌站搭輕軌到T-mobile球賽大約30分鐘左右，車站有售票機器可以買單程票，也可以買Ocra Card用儲值的方式搭乘，車廂蠻乾淨也沒有臭味，感覺上也還蠻安全的。

蠻推薦不排斥看棒球比賽的人，有時間的話可以在美國看場棒球，氛圍跟台灣差蠻多的，很可以體驗看看！

Day2｜西雅圖市區遊#

🍽️ 星巴克工廠店Starbucks Reserve Roastery → 派克市場Pike Place Marke → 口香糖牆Gum Wall → 🍽️ 蛤蜊濃湯 Pike Place Chowder → 華盛頓大學University of Washington看櫻花 → 🍽️ The Crab Pot → Kerry Park看夜景 → 🏠 Red Roof Inn

Note

早上再次搭輕軌從機場前往市區，先到星巴克工廠店Starbucks Reserve Roastery去參觀加上吃午餐，這邊有烘豆的機器，也有一些解說體驗可以參加，標誌是一個R跟一顆星星圖案，可以在裡面用餐、逛逛。

派克市場即時大概半小時可以逛完，口香糖牆就是拍照打卡的地方，大約留5分鐘即可。可能因為去的時候平日加上下雨，所以有名的蛤蜊濃湯 Pike Place Chowder，我們大概排隊15分鐘左右而已，店內也有自助式的座位可以用餐。

這次剛好碰上櫻花季，所以就特別到華盛頓大學看櫻花，盛開的櫻花樹加上古典建築，真是特別好看呢！

The Crab Pot是在碼頭旁邊的手抓海鮮餐廳，烹飪方式是沒有醬，煮過之後撒調味粉的，我們個人口味是覺得普通，沒有特別推薦。

Kerry Park是在比較富有一點的社區裡，晚上也蠻多人會來看夜景的，所以晚上到這邊體感上蠻安全的，只是若搭乘大眾運輸工具前往，還是要特別留意，不要到太晚。

若是住在市區或要到市區轉輕軌前往機場的旅人，請特別留意Westlake Center輕軌站附近麥當勞門口，下午就開始會有很多奇怪的人聚集，晚上則是變成一大群聚集在那邊，雖然麥當勞那側的Westlake Center輕軌入口會有站警衛，但還是請特別注意安全!!!

Day3｜西雅圖 - 溫哥華 - 溫哥華市區遊#

🍽️ Zeitgeist Coffee → 🚌 SEA(Seattle King Street Station) - YVR(Pacific Central Station) → Main Street–Science World Station → Bounce Luggage Storage寄行李 → 🍽️ Straight Brooklyn Pizza → 蒸汽鐘 → 加拿大廣場Canada Place → Burrard SkyTrain Station看櫻花 → 史丹利公園 → 格蘭維爾島市場Granville Island Public Market → 🍽️ suka豬排 → David Lam Park夜櫻 → 溫哥華機場

Note

一早再次搭輕軌從機場前往市區，先到車站Zeitgeist Coffee享用早餐，這間咖啡廳的風格感覺很溫馨，咖啡也很好喝。接著到King Street Station等待搭車，本來是購買Amtrak火車票前往加拿大，結果在前一天收到通知說火車設備異常，要更改成巴士。所以我們就改成搭巴士的方式入境加拿大，在兩國之間邊境處，巴士會停下來讓所有乘客到海關處檢查護照後，再讓大家從新上車到溫哥華Pacific Central Station。

抵達後，我們走到附近的Main Street–Science World Station買Day Pass，除了水上計程車應該是其他大眾都可以搭乘，只要進站跟上公車時逼卡即可。我們搭到Waterfront去Bounce Luggage Storage寄放行李，這是一間商店櫃檯裡面存放行李而已，不是專門的寄放處，但價格、商店位置還不錯以及我們寄放一天也沒有啥問題發生。

前往主要的鬧區，先去吃美國品牌Straight Brooklyn Pizza，有內用區，是單片賣且屬於薄的餅皮，很香又好吃，推薦！吃完後散步到蒸汽鐘，每15分鐘會蒸氣鳴笛。這附近也很多家紀念品店可以逛。可以沿路散步到加拿大廣場Canada Place看一下白帆屋頂建築。

Burrard SkyTrain Station這一站出口一上到地面上就能看到整排的櫻花樹人行道，很有日本的風格哈哈哈。接著搭車到史丹利公園，這邊的櫻花樹還沒全開，不過整個公園很適合散步跟野餐！

從史丹利公園前往格蘭維爾島市場Granville Island Public Market需要轉一次公車，不過算蠻方便，公車班次也很多。這邊就蠻熱鬧的，大家來這市集可以買水果!!!這次我們有買了藍莓跟綠葡萄都超新鮮又甜又便宜，整個超划算!!!市集裡面也有一家貝果店Siegel’s Bagels蠻好吃的，推薦!!

因為是跟朋友約，所以就跟著朋友一起去吃日式豬排店，這家吃起來蠻道地的，也推薦！但我自己是有查到市集附近有一家Go Fish炸魚店，大家有機會的話也可以去試試。

David Lam Park夜櫻，據說是每年都會舉辦的活動，大家若是三月底四月初前往溫哥華旅遊的話，可以查一下這個活動的舉辦時間，晚上會在櫻花樹上打燈，這邊也有一些攤販，有時間的話也蠻推薦過來這邊逛一下。

因為隔天是清晨的飛機，所以我們就直接到機場去等報到了。

Day4｜溫哥華機場 - 卡加利機場 - 黃刀鎮機場(延誤)#

✈️ YVR - YYC - YZF → 🏠 Westin

Note

溫哥華機場整體有很多加拿大特色的裝置藝術蠻酷的，機場感覺上也很溫馨，不愧是北美數一數二的機場，國內線主要分西捷跟加航兩區，指示都蠻清楚的，這邊感覺比較流行用自助報到機，在幾台上掃護照，點一點登機證跟行李條就印出來了，貼好行李後，前往行李託運去，就完成了。

西捷航空我們是買一般的票，是只有含個人物品也就是放在座位底下那種，實際尺寸可以到官網確認，沒有包含手提跟托運行李!!，買票的時後要記得看清楚，然後我們大約是在起飛前一週在官網上輸入定位編號預購托運行李的，如果有要買托運行李請儘早購買，越靠近起飛日期，好像會變貴！

第一段航程出發到卡加利機場都蠻順利的，機上也會發飲料跟小點心。結果！我一下飛機就收到第二段航程延誤到明天的Email，直接衝去櫃檯申訴要改其他班，結果就是完全沒有辦法改，櫃檯就請我們還是等到隔天早上再飛QQQQQQ 直好前往機場附近的酒店休息跟用飯店設施。

Day5｜卡加利機場 - 黃刀鎮機場 - 超市採買 - 追極光#

✈️ YYC - YZF → 極光團旅行社接機 → 超市採買 → 🍽️ korea house → 追極光 → 🏠 Nova Inn

Note

旅行社接機後，就先將我們載到飯店休息整裝，然後我們就到市區去吃韓式料理跟超市採買，接著就回飯店補眠，等待晚上10點出發的追光之旅！今天是安排開車追光，領隊大約換了四個地點，但是今天雲層很厚，所以沒追到光QQQQ

Day6｜市區遊 - 追極光#

狗拉雪撬 → 🍽️ 泡麵 → 市區遊：大奴湖冰雪城堡 → 飛行員紀念碑Bush Pilots Monument → 遊客中心Yellowknife Visitor Centre → 歡迎紀念碑Welcome to yellowknife → 古蹟博物館Prince of Wales Northern Heritage Centre → 🍽️ Bullock’s Bistro → 追極光 → 🏠 Nova Inn

Note

中午導遊先接送我們到湖上參加自駕式狗拉雪撬行程，大約是2-3人一台，我們剛好是2人一台，一位站著踩著雪撬另一位舒適地坐在前座裡，會有一位專員騎雪上摩托車在狗狗前面開路，站在的人也不吃力，狗狗其實會自動跟著專員前進！今天剛好碰上下雪，所以整個更有體驗感～～～

整個行程大約30分鐘左右，繞湖一圈就回到停車場接送回飯店，因為沒隔多久就要參加下午的市區遊導覽，所以我們就在飯店吃泡麵當午餐，然後午睡一下。

下午導遊先帶我們到大奴湖欣賞冰雪城堡，這是黃刀鎮每年都有的慶典活動，不過應該是到3/28而已，所以我們只有到城堡外面拍拍照，參觀完就剛好遇到主辦單位來拆冰雕了，原本還以為會放到春夏讓他自動融化勒XD

接著是到飛行員紀念碑Bush Pilots Monument 這邊會爬一段大概100階左右上到觀景台，觀景台可以看到整個舊城區的！拍拍照我們就下來繼續前往遊客中心Yellowknife Visitor Centre，這邊可以領北緯60度證書以及一張附有小黃刀別針的明信片，這是不跟團也可以自行前往領取的！進到遊客中心，櫃檯人員就會拿給你，也有提供很多黃刀鎮特色動物的印章，可以自己蓋在證書或明信片上！

歡迎紀念碑Welcome to yellowknife這邊當然是必打卡點，旁邊有一個迷你步道可以爬上去看以前負責運送西北地區物資的飛機。接著到古蹟博物館參觀跟聽導遊介紹。結束後請導遊幫我們載到這邊有名的烤魚店Bullock’s Bistro，我們是平日大約接近晚上6點來，大概等10分鐘就入座，這邊也5-6種當地的魚種，然後可以自己選擇炸、烤、煎三種烹飪方式，每份都會附有薯條跟沙拉，上餐速度算蠻快的，魚吃起來很新鮮，但是我吃不出跟台灣的魚有啥咪差別哈哈哈哈

今天本來旅行社是安排帳篷等極光，但因為我們是最後一晚加上昨天沒看到，所以導遊Tony建議我們跟著他開車追光，我們當然馬上同意，結果Tony真的帶我們追到極光拉!!!!親眼看到的時候，真的超感動，有種此生無憾的感覺，極光真的太酷拉!!!!

Day7｜黃刀鎮機場 - 卡加利機場 - Canmore小鎮#

🍽️ a taste of saigon → 送機 → ✈️ YZF - YYC → 卡加利機場租車 → Canmore小鎮 → independent超市採買 → 🍽️ A&W漢堡 → 🏠 MTN House By Basecamp

Note

今天早上退房後，可以先將行李跟加拿大鵝衣物寄放在飯店，然後我們到市區寄買明信片後到越南料理店吃午餐，接著就回飯店等旅行社送機拉！黃刀鎮機場很小，安檢也很快，所以我們大概是起飛前1.5h才抵達。

抵達卡加利機場後，大概步行5分鐘就能到租車的櫃檯，我們這次是訂thrifty，服務、車況整體都還不錯，之前去美西也有租過hertz，這兩家我們都沒遇到什麼問題，然後卡加利機場Hertz、thrifty、dollar三間櫃檯是在一起的，所以感覺大家可以用價格作為考量因素。

因為擔心天氣因素跟怕飛機延誤，所以今天我們就只有先開到Canmore這個小鎮，大約開40分鐘左右，沿路一直開楓葉1號高速公路就可以拉。Canmore小鎮整個超可愛!!!除了房屋的顏色都很好看之外，斜屋頂加上前幾天有下過雪在配上背景的雪山，整個看起來就像是薑餅屋這樣的童話小鎮!!!!小鎮上機能也很好，有超市、餐廳、連鎖速食店！

Day8｜Banff國家公園#

前往Banff國家公園 → 買國家公園門票 → 🍽️ Juniper早午餐 → 弓湖Bow Lake → 天然橋 Natural Bridge → 翡翠湖 Emerald Lake → 露易絲湖 Lake Louise → 🏠 Fairmont Château Lake Louise

Note

一早退房後就開車前往Banff國家公園拉，我們是在前往Juniper早午餐的路上，經過一個像是收費站的地方購買的入園通行證，一台車好像可以買團體票(超過2位的話就會比較划算了)，買好後窗口會把通行日期貼上膠帶讓你貼在車窗上。

Juniper早午餐是飯店的餐廳，風景還不錯，餐點也蠻好吃，是在Canmore開往國家公園的路上。享用完早餐後就先前往這次安排最遠的點弓湖Bow Lake，因為都有鏟雪車將雪移至兩旁，所以沿路都蠻好開的，只有彎進去Bow Lake停車場的一小段路上還有些滑冰，不過速度慢一點也沒啥問題。因為來的時候是四月初，所以整個Bow Lake都還是結冰的狀態，拍照起來很酷一大片雪白配山景，生在緯度低的地方真的很難想像整片湖都結冰甚至可以上去走～～～

天然橋 Natural Bridge跟翡翠湖 Emerald Lake是位在幽鶴國家公園(班夫開著開著就到幽鶴了)，天然橋是在去翡翠湖的路途中的一個景點，是一座由河水常年累月的沖刷而成的橋樑，算是一道自然奇觀，河水是藍綠色的，很清澈。翡翠湖 Emerald Lake的話，則因為季節的關係也還是雪白一片，不過旁邊有深綠色的樹搭配，拍起來也很美麗。

最後就是今天的重頭戲：露易絲湖 Lake Louise拉！我們是先開車到Lake Louise的停車場，在走到露易絲湖上拍照，已經有很小範圍開始融化了，不過整體還是不影響走到湖上散步。隨後我們就到今日最期待的飯店Fairmont Château Lake Louise去check in拉！開到飯店門口之前的右邊會有一個Enter的符號，那邊就是停車場，右轉進去取卡停好車後，就拿著那張卡跟行李到櫃檯報到。

我們是訂湖景房，有一個窗戶正對著湖中間的山，非常漂亮，窗邊邊有擺一組沙發可以在這邊泡咖啡吃小點心，看風景發呆很舒服！傍晚時，我們有參加湖邊一個烤棉花糖的活動，還蠻不錯的！不過因為太陽下山後蠻凍的，所以我們就趕快回飯店去泡spa池了！飯店內有餐廳，有預算的旅人可以到餐廳享用早午晚以及下午茶，沒預算的話，可以像我們一樣買一些超商的即時食品過來吃。

大概十點左右，我們有開車到Lake Louise的停車場去拍星星，這邊光害比較小，星星超級無敵多，iphone就能直出超美麗的相片～！

Day9｜Banff國家公園 - 班夫小鎮 - Banff Gondola - 弓河瀑布 - 驚奇角#

morant’s curve → Castle Cliff Viewpoint → Johnston Canyon → Vermilion Lakes → Banff sign → 🍽️ the old spaghetti factory → 班夫小鎮 → 🍽️ Cows冰淇淋 → Banff Gondola → 弓河瀑布Bow Falls → 驚奇角Surprise Corner → 🍽️ wendy’s → 🏠 Grande Rockies Resort-Bellstar Hotels & Resorts

Note

飯店也可以加購早餐的選項，不過因為房間的風景跟餐廳看的是一樣的，所以早上我們就到飯店樓下的輕食店，買了咖啡跟輕食回房間坐在窗邊看風景享用早餐！退房後，就繼續前往幾個有名的景點拍照打卡。

morant’s curve是看山水火車的一個風景點，不過會不會剛好遇到火車就很看運氣了，但整體拍照起來也是蠻漂亮的。Castle Cliff Viewpoint則是剛好可以看見Castle Mountain的峭壁。接著到Johnston Canyon強斯頓峽谷，這邊主要欣賞峽谷及瀑布，但離停車場必須健行一小段路，不過這邊的步道整理的非常好走，這裡有兩座瀑布Lower and Upper Falls，從停車場大約1.1公里單程大概走30分鐘可以抵達Lower Falls。繼續往前走的話，可以到Upper Falls，這邊就比較看體力，大約2.6公里單程大概走1小時。 !!如果是四月初來的話，還是需要冰爪套在鞋子上會比較安全(冰爪我們是買蝦皮上一雙50塊左右拋棄式的)!!

回班夫的路上經過Vermilion Lakes一樣也是白白一片湖，所以拍個照就繼續行程了。Banff sign對面有停車場可以停，不過因為是在道路旁，不會有路人經過，所以可能需要自備腳架，不然就只能等待其他遊客也剛好來拍照才能互相幫忙拍了！

the old spaghetti factory是連鎖的義式店，因為點一份主餐就會含湯、麵包、咖啡/茶、冰淇淋，所以CP值很高，如果最後旅費拮据的話，還是可以去吃一下的，至於餐點味道就是中等，沒有特別驚艷！

班夫小鎮上有很多店家可以逛，加拿大鵝、始祖鳥、lululemon、patagonia、The North Face等等品牌可以逛，不過感覺是沒有特別便宜。不過這裡的Roots有Banff紀念衫，也有吊飾小海貍穿著Banff的衣服。

Cows冰淇淋是加拿大的品牌，我們這次是點cownadian maple口味，不會太甜太膩，很濃郁蠻好吃的！BeaverTails也很多人推薦，不過這次沒機會吃到，下次肯定要試試看！

Banff Gondola我個人很推薦一定要去，由於我們這次不是旺季，所以我們就現場才買票，如果時間已經確認好的旅人，也可以先線上購票，因為他價格是浮動的，官網上越靠近日期也會開始變貴！纜車的車廂蠻小的，感覺坐4位大人應該差不多了，整趟大概8分鐘，上升幅度真的蠻陡，除了搭纜車上升也可以爬步道(但好像要爬5公里，因為步道是繞著山蜿蜒走)，所以搭纜車上山的時候，就能看下底下有人健行中。

抵達山頂後戶外景觀台有看板介紹每座山的名字，也有一個來回2公里的木棧步道是通往氣象觀測站，步道蠻平坦，沿著木棧道走走，走累了也有板凳可以休息、觀景台可拍照，天氣好的話蠻推薦散步一下的，因為不是旺季，所以下山的時間也沒有受到限制，等我們散步回來後就拿原本的票給工作人員看，就可以搭纜車下山了。山頂山腳都有紀念品店，賣的商品差不多價格也差不多，山腳有星巴克，山頂上也有咖啡店。

弓河瀑布 Bow Falls直接停車場旁邊就是觀賞點了。驚奇角Surprise Corner可以遠觀班夫溫泉大飯店與弓河河景！接著我們就回Canmore小鎮買wendy’s當晚餐，然後回飯店休息了！

今天因為極光kp指數有到6，所以晚上我們有開往山上停在路邊等極光，肉眼看到很微弱，不過手機可以直接拍的出來！大家去班夫的時候還是可以看一下極光預測網站，說不定有意外的驚喜。

Day10｜CrossIron Mills outlet - 桃園機場 - 卡加利機場還車 - 西雅圖機場#

🍽️ Rocky Mountain Bagel Company → 🛍️ CrossIron Mills outlet → 🍽️ Jollibee → 卡加利機場還車 → ✈️ YYC - SEA

Note

Rocky Mountain Bagel Company的停車位很少，所以如果是住在附近建議可以早上散步過來買，餐點製作還蠻快的，可以先到櫃檯左邊看一下各種口味的外貌來決定點哪個口味！

接著大約開車1小時15分鐘抵達CrossIron Mills outlet！可以先到服務台拿護照領一張Vip卡，不過不一定每間店都能用。另外，想要特別推薦一家mountain warehouse是之前我們有買過他的高領抓絨衣，材質穿起來很舒服又很保暖，而且一件才台幣400！

卡加利機場還車的動線很不錯，一樣是往機場方向開，會有還車的指示，把車開回停車場後就找到你租車公司的位置，停好後會有專員來跟你核對資料跟檢查車輛外觀，一切都沒問題後，就可以推行李走到卡加利機場拉！

西雅圖機場只有兩家店可以吃，都是炸物類型的店，可以等一開櫃就把行李托運，然後入關去裡面找食物吃，裡面比較多選擇！

Day11｜西雅圖機場 - 桃園機場#

總結#

如果目標是黃刀鎮或班夫，我比較推薦飛到溫哥華轉機，航班比較多機票也便宜一點！

極光團我是直接在 Aurora Dream Tour極光夢之旅的官網上預訂，官網上有各種天數、是否含住宿的套裝行程、自費行程，全部都也有用中文列出來，非常簡顯易懂，官方客服回訊息速度也很快！

我們這次是選中等酒店的四天三夜套裝行程，入住的是Nova Inn飯店，飯店優點是地理位置好，可以走路到市區餐廳、超市採買、wifi速度也還不錯，缺點是飯店設備不新，隔音普通，如果比較在意的話，建議是可以選升等飯店的套裝，因為冬天去的話，外面蠻冷的，除了市區遊跟自費設施，待在飯店時間會蠻多的。

這次的溫度大概是-8 ~ -20°C左右，我們裡面就是穿UNIQLO的超級暖發熱衣+戶外用品店賣的半拉鍊微抓絨長袖+The North Face三合一內裡的羽絨外套，下身的話就是UNIQLO的超級暖發熱褲，然後外面再穿極光團含的加拿大鵝服裝是蠻夠的，但是晚上追極光站在戶外久了，其實還是體感還是蠻凍的。

可以帶腳架，如果有幸運看到極光，iphone長曝光就能拍出極光了！我們的套裝也有含領隊專業相機的拍照！

極光團的時間我有特別挑在新月附近，據說是比較不會被月亮亮度影響到，然後我們當天的kp值預測大概是3左右，但已經能看到跳動的狀態了！

現金的話，在加拿大幾乎是用不到的，這次除了極光團的建議小費給現金，跟我們個人習慣加油用現金之外，其他完全都可以刷卡！小費餐廳的機器上大概都是預設寫15%-22%！
