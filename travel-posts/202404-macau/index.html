🇲🇴 澳門

1 min

Time：2024/04#

✈️ 航班與入境#

去程：RMQ → MFM
回程：MFM → RMQ

🗓️ 行程#

Day1｜台中機場 - 澳門機場 - TOMAMU - 戀愛巷 - 大三巴牌坊 - 議事亭前地#

台中機場 → 澳門機場 → 住宿check in → 戀愛巷 → 大三巴牌坊 → 🍽️ 咖哩榮魚旦牛雜 → 🍽️ 澳門老酸奶 → 🍽️ 義順牛奶 → 議事亭前地 → 🍽️ 甜香園 → 🍽️ 贏到粥 → 🏠 Belive Andmore

Note

抵達澳門後，我們先搭計程車到飯店放行李，接著搭公車到戀愛巷，單趟投幣的話是6澳門幣/人，戀愛巷往上走就是大三巴牌坊，然後牌坊前面的廣場和小巷子就有許多的食物店與伴手禮店，可以沿路吃吃喝喝買買買到議事亭前地。因為我們住宿離這邊不遠，所以就順路散步去買甜香園，再慢慢走回飯店，休息片刻後，就出發去吃贏到粥拉(這家可以刷卡)～必點卜卜蜆～很香!!!

Day2｜官也街 - 飯店巡禮 - 永利皇宮纜車 - 美獅美高梅#

🍽️ 旺記咖啡 → 龍環葡韻 → 🍽️ 安德魯蛋塔 → 官也街 → 🍽️ 澤賢記蛋球仔 → 飯店巡禮 → 永利皇宮纜車 → 美獅美高梅 → 🏠 Belive Andmore

Note

早餐先搭公車到旺記咖啡，二樓有用餐區，坐下之後跟服務員點餐就可以了，蠻好吃的，不過這家只能用現金付款喔!!吃飽後大概走10分鐘到龍環葡韻住宅博物館，是免費入內參觀的，大概15分鐘看一看過去人家的房子裝潢擺設。

回到官也街吃吃喝喝以及買安德魯蛋塔(這家可以刷卡)，站在門口就可以聞到滿滿的蛋塔香～然後走去澤賢記蛋球仔，是裡面包有珍珠的雞蛋仔，蠻香蠻好吃的～

搭公車到威尼斯人，入內逛逛商店街、人造遊船、唱歌表演～然後搭公車去永利皇宮看水舞秀以及搭乘永利皇宮的纜車，搭乘是免費的，可以繞水池一大圈回到原點，也可以只搭半程到永利皇宮下車。

最後安排來澳門大部分人都會體驗的賭場～進去時檢查人員可能會查驗證件看年紀是否已經21歲，進去後，美獅美高梅有提供簡易的點心飲料吧可以邊玩邊享用，機台的話，可以選有提供小額的角子機體驗一下氛圍，然後要記得裡面是不可以拍照的喔！

Day3｜澳門旅遊塔 - 漁人碼頭#

🍽️ 瑪嘉烈蛋塔 → 🍽️ 澳門酸辣粉 → 澳門旅遊塔 → 漁人碼頭 → 🍽️ 媽閣銘記澳葡美食 → 🏠 Belive Andmore

Note

起床後搭公車來吃瑪嘉烈蛋塔(這家無法刷卡)，聽說這家是跟安德魯分家後開的，兩家各有風味，建議有時間的旅人，兩家都可以來嘗試看看～都蠻好吃的～～

距離我們預約的澳門旅遊塔還有一點時間，所以就在附近逛逛，意外找到一家澳門酸辣粉，蠻道地也蠻好吃的，不過座位偏少，然後只收現金喔。因為太無聊，所以晃到新葡京賭場去看看有何不同～

然後搭新葡京的接駁車來澳門旅遊塔報到，我們是買klook的二合一行程包含下午茶及景觀台門票，因為有用餐時間限制，所以我們先往360度旋轉餐廳的指標去報到及用餐，用餐完畢之後搭電梯到一樓再往景觀台的指標方向，搭電梯上去觀景台，這兩個地方的電梯入口是在不同的地方，要根據自己買的票券方案，安排前後順序。下午茶套餐是一人一份已經配好的小食及點心，只有飲料是自助吧，繞一圈是90分鐘，繞的速度蠻慢的，邊吃飯邊看風景是沒問題的，不過餐點就是普通，沒有特別好吃。

搭公車來到漁人碼頭，不知道是不是平日，所以商店街沒什麼開也沒什麼遊客，不過傍晚時來這邊散步逛逛，蠻Chill的，也不太熱～

回到飯店附近的媽閣銘記澳葡美食(這家可以刷卡)，大概排隊了15分鐘，點了乾免治牛肉飯、馳名燒春雞、西烊煏牛尾，都蠻好吃的，這家大推!!!

Day4｜澳門機場 - 台中機場#

澳門機場 → 🍽️ 麥當勞 → 台中機場

Note

一早搭公車抵達澳門機場，櫃檯報到後，到樓上的麥當勞享用早餐～這邊的麥當勞有扭扭粉(像是水煮通心粉)，還有港奶可以點，推薦大家來嚐嚐～～

總結#

澳門不大，可以分為北邊的澳門島和南邊的氹仔島，搭公車都還算方便，四月去白天有點悶熱，但是到晚上就比較涼爽了！這趟遇到蠻多小吃店不能刷卡，建議大家還是可以多換一些現金備用，畢竟澳門很容易就會被小吃給吸引呀～～

3. 玩樂 TWD 815 / 人
